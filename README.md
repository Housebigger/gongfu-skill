# gongfu-skill

这里的 `gongfu`，取“共富”之意。

这是一个以“促进广大劳动人民具备零成本创业的意愿和条件”为宗旨的知识仓库。它更关心的，不是个人流水账式记录，而是那些能够被反复调用、持续积累、低门槛传播的方法论、认知框架、经验结晶与伴随式反思。

它也明确服务于一种立场：新技术、新工具、新生产力，不应只成为资本垄断和加剧不公的手段，而应尽可能转化为普通劳动者可以掌握、可以使用、可以据此改善命运的现实条件。

## 仓库宗旨摘要

- 本仓库首先服务于一个目标：让广大劳动人民逐步具备零成本创业的意愿、认知、方法与起步条件。
- 王侯将相宁有种乎，荣华富贵人人有份；普通人也应当有机会凭借知识、工具、协作与行动，去争取更好的生活与更大的发展空间。
- 机器人、人工智能等新生产力，应尽可能服务于劳动者能力提升、低成本创造价值和更广泛的社会普惠，而不应沦为资本垄断和加剧不公的工具。
- `methodology/` 负责沉淀可复用的方法与主题化认知，`accumulation_settle/` 负责沉淀跨文章经验结晶与伴随式反思。

如果你想先快速理解这个仓库为什么叫 `gongfu-skill`，先读这一节；如果你想进一步理解这个仓库准备先做什么、再做什么，可以继续看下面的“阶段性目标”；如果你想读更完整、更鲜明的表达，再继续看后面的《共富宣言》。

## 阶段性目标

这三阶段不是三段彼此分离的口号，而是一条前后相接、层层递进的共富路线：先完成思想武装，再走向现实建功，最终实现技能共富，让劳动者的觉醒、行动、创造与服务能力一步步汇聚成面向大众、照亮万家的新型生产力。

### 第一阶段
先把伟人留下的思想武器，从书斋里的概念、神坛上的赞叹和口头上的敬仰之中，重新交还到新一代劳动人民手中；以最低门槛、最低成本、最少负担的方式，让普通人也能学得会、拿得起、用得上，在各行各业中重新认识现实、重新校准判断、重新组织行动、重新创造价值，让劳动者先拥有看世界、做事情、开新局的思想底盘。

### 第二阶段
再把已经被劳动者真正掌握的思想力量，从个人觉醒和局部实践，进一步推向新时代中国的发展主战场；紧扣“十五五”规划所展开的发展方向、产业任务与时代机遇，为身处各个行业的劳动人民提炼可理解、可执行、可落地的现实指引，使更多普通人不仅能看清趋势、站稳脚跟，还能顺势成长、借势创业、结成协作、开辟新业，在时代洪流中为自己、为家庭、为社会共同做出新蛋糕。

### 第三阶段
最终把分散在百行百业、沉睡于岗位之间的专业智慧，进一步转化成“技能上架、才华流通、智慧共富”的新型技能生产关系：让每一位有思想、有业务理解、懂行业底层逻辑、摸过真实流程与关键难点的专业人才，都有能力把自己多年积累的知识、方法、判断、流程、模板与经验，蒸馏成一个个可调用、可验证、可组合、可交易的 `skill`；再把这些 `skill` 挂牌到各类乃至各大 agent-skillhub 平台之上，让原本锁在岗位、公司、地域与时段里的专业本领，从一次性的体力与脑力消耗，转化为可持续供给、可被反复调用、可被公平租售的电子服务。如此一来，真正懂行的人不必再把才华困在简历和工位里，普通家庭也不必再隔着高门槛才能接触高水平服务；百行百业的经验、诀窍与判断，将以更低成本流向社会、进入市场、服务大众，既让专业者获得应有回报，也让万家灯火共享数字时代的新型生产力，最终推动形成“人人能沉淀技能、人人能出售本领、人人能用知识创造共富”的新局面。

## 共富宣言

共同富裕从来不是遥不可及的远景规划，更不是脱离现实的空谈幻想，时代发展走到今天，已然步入理念凝聚、价值共建、利益普惠的关键窗口期。广大劳动人民更应清醒认知：共同富裕从来不会凭空降临，既不能被动寄望于资本群体的良心自省与善意施舍，也不能坐等所谓天降青天大老爷的怜悯成全、自上而下的单向安排。我们更当传承祖祖辈辈薪火相传的勤劳本色，执起耕耘大地的镰刀锄头，握紧建设家园的锤子铁锹，以踏实立身去开垦时代沃土，以勤勉奋斗去创造美好生活，靠自身双手打拼属于自己的幸福与公平。

机器人行业，是为数字世界筑牢实体根基、搭建智能躯体的基础性行业，是赋能社会生产、大幅拉升生产力与生产效率的关键性行业，更是颠覆传统劳作模式、重塑劳动形态、解放人力束缚的前瞻性行业。承载着科技变革与产业升级使命的机器人领域，其核心技术、产业链优势与时代发展红利，绝不能任由大型资本肆意垄断、私自把控，更不能沦为资本无序扩张、固化阶层差距、开展新一轮劳动剥削的冰冷凶器。

与机器人行业相辅相成的人工智能行业，更是数字时代的“智慧大脑”，是重构生产生活方式、激活全产业链活力、推动生产力迭代升级的核心引擎。从智能算法的精准赋能到大数据的高效应用，从人机协同的高效生产到民生服务的便捷升级，人工智能的每一步发展，都与广大劳动人民的切身利益紧密相连，都承载着解放人力、提升福祉、推动社会进步的时代使命。这一行业的核心算法、数据资源、技术成果，同样不能被大资本所垄断、所操控——若任由资本将人工智能异化为精准压榨劳动价值、收割大众利益、固化阶层壁垒的工具，不仅会背离科技发展的初心，更会阻碍共同富裕的推进步伐。人工智能的智慧之光，理应照亮每一位劳动者的奋斗之路，其技术成果理应归全体人民共建共享，成为助力劳动者提升能力、减轻负担、实现价值的强大支撑，而非资本牟取暴利、加剧不公的手段。

新时代能够解放生产力、引领产业变革的新型生产工具、科创成果与核心生产资料，理当扎根人民、归属人民、服务人民，由广大劳动者共同参与创造、共同享有红利、共同守护成果。那些漠视劳动法规底线、钻制度空子大肆敛财的逐利之徒，那些脱离群众根基、高高在上骑在百姓头上作威作福的傲慢者，那些身处社会主义时代语境，却以资本权贵自居、沾沾自喜凌驾于大众利益之上的利己者，那些依附公共资源、吸附社会根基、把普通民众当作牟利血包的寄生蛀虫，终将被时代进步的洪流涤荡净化，被社会公平正义的大势冲刷淘汰，更会被滚滚向前的历史车轮碾作尘埃。

生而无产，立身以勤；生逢时代，守心以正。我辈劳动者当觉醒本心、自立自强，不甘平庸、不愿受制，以奋斗守护尊严，以实干构筑梦想，众志成城齐心捍卫人人共建、人人共享、人人共富的美好未来。起来，不愿做奴隶的人们。
