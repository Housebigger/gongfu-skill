# accumulation_settle

这个目录存放的不是“单篇原文 -> 单篇转译”的正文，而是做完一轮工作之后留下来的沉淀层材料。

它当前分成两块：

## 1. `experience_crystallization/`
用来存放跨文章、跨任务、跨主题的经验结晶。

当前实际保留：12 个 markdown 文件
- `excr001-mao-study-execution-wisdom.md`
- `excr002-theme-summaries-index.md`
- `101—108` 八个主题化经验汇总
- `excr011-theme-summary-source-mapping.md`
- `excr012-third-volume-final-victory-window-stage-summary.md`

适合在以下情况下进入：
- 你不想先看单篇文章，而想直接看抽象出来的方法线
- 你想看“识势定向 / 统一战线 / 主体建设 / 危机治理”这类跨文章母题
- 你想从执行经验而不是文本顺序来反查方法

## 2. `spiritual_growth/`
用来存放伴随式的自我描述与成长记录。

当前实际保留：2 个 markdown 文件
- `000-initial-persona-self-portrait.md`
- `001-growth-journal-during-mao-study.md`

适合在以下情况下进入：
- 你想看这个长期项目如何反过来塑造工作风格和自我理解
- 你想区分“方法论正文”和“陪伴式成长反思”

## 推荐使用顺序

- 想看经验母题：先读 `experience_crystallization/README.md`
- 想看伴随反思：先读 `spiritual_growth/README.md`

## 说明

这个目录只描述当前 `gongfu-skill` 仓库里实际存在的沉淀材料，不默认旧仓库里的其它辅助文件已经一并迁入。
