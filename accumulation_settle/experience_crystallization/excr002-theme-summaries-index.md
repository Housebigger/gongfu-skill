## 二、使用原则

1. 先从主题 summary 进入，抓母题。
2. 需要看历史生长过程时，优先读 `excr011-theme-summary-source-mapping.md` 与各 theme summary 内的“历史 stage_summary 回链”。
3. 每篇主题 summary 后续都可继续补两类索引：
   - 原文索引：哪些毛主席原文支撑这条主题线
   - 历史 stage_summary 索引：哪些旧 summary 提供了中间沉淀
4. 后续注入时不机械搬运旧 summary，而是抽取可复用的方法判断、误区辨析和现实转译。

## 三、推荐阅读顺序

- 如果你现在想快速看全局，优先读：101 -> 102 -> 108
- 如果你现在想补“组织如何长出来”，优先读：103 -> 104 -> 107
- 如果你现在想补“长期推进与内部治理”，优先读：105 -> 106
