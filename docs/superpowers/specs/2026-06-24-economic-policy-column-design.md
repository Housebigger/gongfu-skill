# 设计文档：战略库经济政策追踪专栏（Phase 1 基础）

- 日期：2026-06-24
- 状态：已通过设计评审，待落地
- 涉及层：`strategy/`（知识层）。本期**不动引擎**。

## 1. 背景与目标

战略库现有两条研究根源：十五五规划（`analysis/` 解读，国家宏观规划方向）与 Serenity 产业链分析法（`industry_investment/`，产业链微观卡点）。本设计新增**第三条根源——经济政策追踪专栏**：持续追踪中国国家经济政策的发布，并从近 10 年政策中摸透"政策↔真实经济社会发展"的作用规律。

三条根源互补：**规划=国家要往哪走（宏观方向）｜ Serenity=产业链微观卡点 ｜ 本专栏=政策与现实的作用规律（宏观实证）**。

**分两期（用户已确认）：**

- **Phase 1（本设计）= 基础**：政策追踪台账 + 近 10 年锚点/主线复盘 + 作用规律提炼。
- **Phase 2（另开 spec，不在本期）= 推演**：基于规律库 + 最新政策 + 国际形势 + 国情，推演中国经济未来走向；引擎接入也放到这一期。

> **本轮交付（用户进一步收窄）= 搭建分线骨架**：本轮只把专栏"分线"搭起来并可用——目录结构 + 各 README + `00 方法论框架`（含劳动者视角翻译）+ `01 政策追踪台账`（近 10 年锚点骨架，逐年核心定调）+ `05 边界与诚实`，以及 `references/policy_archive/` 的信源脚手架（README + 信源清单 + 1—2 份样本锚点）。`02/03 主线深度复盘`、`04 作用规律总结`的全量内容与批量语料整理，列为**后续扩展轮次**（仍属 Phase 1，分批填充）。骨架阶段：`economic_policy/README.md` 用"已建 / 待扩展"清楚标注哪些文件已就位、哪些是后续，避免 README 描述不存在的文件。

## 2. 已确认的范围决策

- **语料来源 = 先搜后确认**：从公开官方源（中国政府网 / 国务院政策文件库 / 发改委 / 央行 / 财政部 / 国家统计局 / 共产党员网会议专题）检索，落地时先抓样本给用户确认信源与范围，再批量整理。不抓订阅墙后的全量库（如北大法宝）。
- **语料范围 = 锚点 + 主线**（不全量）：
  - **锚点时间线**：中央经济工作会议公报（2016—2025，约 10 份）+ 历年政府工作报告 + 五年规划（十三五 2016 / 十四五 2021；十五五已在 `references/`）。
  - **6 条标志性主线**（各自配"政策意图 ↔ 真实结果"）：供给侧结构性改革(2015)、双碳(2020)、共同富裕(2021)、新质生产力(2023)、货币政策转向（稳健→适度宽松，2024）、财政政策转向（积极→更加积极）。
- **本期不动引擎**：Phase 1 是研究语料，先沉淀知识层。把"政策顺风"信号接进引擎（行业判断 / 趋势前瞻）留到 Phase 2。
- **诚实/中立**：如实记录政策原文与官方/统计局可查的真实结果；政策↔结果按**观察到的关联**陈述，不下严格因果断言；不做政治评判、不做投资建议、剥离个股。

## 3. 可行性与信源（已搜索确认）

- 中央经济工作会议年年有、定调可追（2017"七个坚持"→2018"五个必须"→…→2025"五个必须"）。共产党员网有[历次会议专题存档](https://www.12371.cn/special/lczyjjgzhy/)。
- [国务院政策文件库](https://sousuo.www.gov.cn/zcwjk/policyDocumentLibrary)（国办主办、公开可检索）为政策原文主渠道；[中国政府网政策频道](https://www.gov.cn/zhengce/)、[发改委文件库](https://www.ndrc.gov.cn/xxgk/wjk/)为辅。
- "作用规律"可做：官方回顾（如[供给侧改革深度观察](https://www.gov.cn/yaowen/liebiao/202506/content_7026829.htm)）+ [国家统计局](https://www.stats.gov.cn/)公开指标，可把"政策意图↔真实结果"配对（示例事实：2013—2021 单位 GDP 能耗累计 -27.5%；2016—2021 战略性新兴服务业营收年均 +13.5%；十八大以来普惠型小微贷款 / 涉农贷款年均增速 25.5% / 14.9%）。

## 4. 知识层结构（方案 A：镜像现有根源模式）

### 4.1 原文源料 —— `strategy/references/policy_archive/`

`references/` 定位"原文源料，不加工"。Phase 1 以公开官方源为主，每篇标注来源 URL+日期。

```
strategy/references/policy_archive/
├── README.md            # 定位 + 诚实声明（公开官方源为主，标注来源）
├── 00-信源清单.md       # 锚点文件 + 官方回顾 + 统计口径来源，逐条 URL+日期
└── NN-<文件>.md         # 锚点文件原文/官方摘要一文一档（中央经济工作会议公报、政府工作报告要点、五年规划要点）
```

### 4.2 专栏（新增第 11 个子层）—— `strategy/economic_policy/`

镜像现有子层编号约定（`00` 恒为方法论框架）。

```
strategy/economic_policy/
├── README.md                                  # 专栏定位 + 两期规划 + 与 analysis/perspective 分工
├── 00-方法论框架.md                            # "政策怎么读、怎么传导到真实经济"分析法 + 劳动者视角翻译（核心）
├── 01-政策追踪台账.md                          # 近10年锚点时间线（会议/报告/规划，每条：时间/文件/核心定调/关键提法变化）
├── 02-政策主线复盘（上）-供给侧·双碳·新质生产力.md  # 政策意图 ↔ 真实结果
├── 03-政策主线复盘（下）-共同富裕·货币转向·财政转向.md
├── 04-作用规律总结.md                          # 核心交付：政策时滞、传导机制、见效快慢、对就业/收入的传导规律
└── 05-边界与诚实.md                            # 规律复盘不是预言；中立、可追溯、关联非严格因果
```

### 4.3 劳动者视角翻译（专栏的灵魂，主要写进 `00`）

| 政策研究的常规用途 | 共富·劳动者版 |
|---|---|
| 读懂政策定调 | **看国家在"扶什么/卡什么"**：政策风口往哪吹，哪些行业/区域/岗位有顺风 |
| 政策有时滞 | **提前布局**：政策出台到见效有时间差，劳动者怎么卡在见效前 |
| 政策↔真实结果规律 | **别只听口号看落地**：哪些政策真传导到就业/收入，哪些雷声大雨点小 |
| 政策主线演进 | 呼应方法论库的**实事求是 / 抓主要矛盾**：跟住主线、不追噪声 |

## 5. "作用规律"研究方法（`04` 的做法）

对每条主线：①政策意图（原文定调）→ ②落地举措 → ③官方/统计局可查的真实结果（带数据与来源）→ ④观察到的规律（时滞多长、靠什么传导、对哪类岗位/收入有感）。规律以**观察性关联**表述，明确不等于严格因果（多因素、反事实不可得）。最终在 `04` 汇总成若干条可复用的"读政策"规律。

## 6. 与现有层的分工（写进 README，避免重叠）

- `analysis/`：读十五五规划**单一文本**；本专栏追踪**政策流**并做历史规律复盘。
- `perspective/`：**行业主题**的中长期前瞻（"方向不预测时间表"）；本专栏是**政策驱动的宏观实证**，且 Phase 2 才做推演。
- `industry_investment/`：产业链**微观**卡点；本专栏是**宏观**政策视角。

## 7. 验证方式（无自动化测试，doc-based）

1. 文件齐全：`references/policy_archive/`（README+信源清单+样本）与 `economic_policy/`（README+00–05）。
2. 台账锚点年份齐（中央经济工作会议 2016—2025 逐年在列）。
3. 每条主线都有"政策意图 ↔ 真实结果"配对，且真实结果带统计局/官方来源链接。
4. `grep -rEn '\$[A-Z]{2,5}\b'` 无个股代码；通读无投资建议、无政治评判。
5. READMEs 与实际文件一致；`strategy/README.md` 子层数 10→11、新增第三根源说明。

## 8. 非目标（YAGNI）

- 不做推演/预测（Phase 2）。
- 不动引擎（无新 YAML、无 router/tools 改动）。
- 不抓订阅墙后全量政策库；不做政策域全覆盖。
- 不出个股/仓位/买卖点；不做政治评判或宣传。

## 9. 待确认 / 风险

- **可抓取性**：gov.cn / 统计局部分页面可能反爬或改版；抓不到的锚点只在信源清单登记 URL，正文用官方可引的公开摘要，不伪造原文。
- **历年定调措辞**：中央经济工作会议 2016 及各年"X 个必须/坚持"的确切框架需落地时逐年核实（搜索已得 2017–2025 主体，2016 待补）。
- **真实结果数据**：以统计局/官方回顾已公布口径为准；指标选取need与政策主线对应，避免挑选性偏差（在 `05` 声明口径与局限）。
