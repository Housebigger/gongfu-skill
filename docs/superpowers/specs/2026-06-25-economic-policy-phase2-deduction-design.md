## 1. 背景与目标

经济政策追踪专栏（战略库第三条研究根源）Phase 1 已完成：政策追踪台账（01）+ 六主线"政策↔真实结果"复盘（02/03）+ 作用规律总结（04）。本设计是 **Phase 2 = 推演**：基于规律库（04）+ 最新政策（01）+ 国际形势 + 国情，**推演中国经济未来的发展方向和经济形势**，并翻译成一线劳动者可用的前瞻判断。

**本轮范围（用户已确认收窄）= 仅推演内容（知识层）。** 引擎接入（把推演方法/方向接进 `gongfu_consult`）留作**下一轮单独 spec**——因为"把预测硬注入引擎"需谨慎设计（更可能注入"方法框架"而非"具体结论"）。

## 2. 已确认的方法决策

- **方法 = A 情景化议题推演**（用户已选）：一个推演方法框架 + 针对几个关键前瞻议题各做基准/上行/下行三情景推演。**每个议题的推演调用 Phase 1 规律库（04）外推**，而不是凭空预测——把"主线规律"当作推演引擎。
- **立场**：保守、情景化；**给方向不给点位/时间表、不做投资建议、不做政治预测**；延续专栏诚实底线，呼应 `perspective/` 的"给方向不给时间表 + 前瞻需持续校验"。
- **时间跨度**：双视角——**近期（2026—2027，1—2 年，较可推）** + **十五五期（2026—2030，方向性）**。
- **国际形势/议题数据来源 = 先搜后确认**：实现时各议题分别在线核实，带官方/权威来源；抓不到的只登记、不伪造。

## 3. 当前形势锚定（搜索已确认，2026 视角；实现时复核更新）

- 2026 是"十五五"开局之年，多家机构预测 GDP 增速约 4.5%—5%；"开好局、起好步"，财政加力。来源：[北大光华《积极扩大内需…2026 中国经济展望》](https://www.gsm.pku.edu.cn/info/1316/31120.htm)、[第一财经《"十五五"中国经济中高速增长，机遇与挑战》](https://www.yicai.com/news/102936303.html)。
- 内需偏弱：居民资产负债表受房价调整收缩、储蓄动机增强，服务消费是 2026 促消费重点。
- "灰犀牛"：房地产、地方债风险；2026 房地产或进一步支持（收储、"好房子"）。
- 外需：美元计出口增速或约 5%，韧性源于贸易斗争边际缓和 + 产业链韧性。
- 国际：中美 2026 进入"战略稳定点"（元首通话），但博弈"常态化、结构化"、"以关税换让步"；2025 关税一度达 145%、对美进出口占比降至 8.8%；全球增长约 2.7%—3.1% 回落，WTO 下调 2026 全球货物贸易增速至 0.5%；供应链重构 / 产业政策民族主义；地缘风险上升。来源：[新华网《2026，世界经济五问》](https://www.news.cn/world/20260103/d58bff34577547bf8c813f69f2288d02/c.html)、[复旦 FDDI《"管理性贸易"：2026 美国对华贸易政策议程》](https://fddi.fudan.edu.cn/c8/91/c21253a772241/page.htm)、[中外对话《2026 国际局势前瞻》](https://www.chinanews.com.cn/dxw/2026/01-01/10544449.shtml)。

## 4. 议题清单（6 个，每个给三情景）

覆盖需求侧、两只"灰犀牛"、供给新动能、外部、结构：

1. **内需/消费**能否成为主引擎（居民资产负债表、服务消费、社保托底）
2. **房地产**何时企稳（灰犀牛、收储、"好房子"供给）
3. **财政可持续与地方债化解**（灰犀牛、"6+4+2"化债、央地关系）
4. **AI + 新质生产力**能否接棒增长（新动能 vs 旧动能退潮）
5. **外需与制造出海**（中美博弈常态化、关税、供应链重构、全球南方）
6. **人口与劳动力**（负增长、老龄化、人力资本红利）

> 设计决策：「关键产业链自主可控」不单列，并入议题 4（新质/科技）与议题 5（出海/供应链重构）——避免与既有议题重叠。

## 5. 每个议题的推演模板（统一）

1. **现状锚点**：议题当前事实，带数据 + 来源（先搜后确认）。
2. **规律库外推**：调用 `04` 的相关规律（时滞 / 传导机制 / 对就业收入传导）作为推演依据，注明"见 04 哪条"。
3. **外部约束 + 国情设定**：套上 `06` 的国际形势 / 国情共用假设。
4. **三情景**：基准 / 上行 / 下行，每个情景给**触发条件 + 关键假设**（情景之间靠假设区分）。
5. **可证伪点 / 观察指标**：盯哪几个指标就知道在走哪条情景（让推演可被现实检验）。
6. **劳动者含义**：每种情景下劳动者怎么准备（行业/岗位/技能/区域的顺逆风与卡位），含负面情形。

措辞为**情景条件式**（"若…则…/在…假设下…"），非断言式；不出个股、不给买卖点、不做政治评判。

## 10. 风险

- **推演的固有不确定性**：情景必然带主观性；以"三情景 + 显式假设 + 可证伪点 + 复盘机制"控制，宁可保守、讲清不确定，不追求"看起来确定"。
- **数据时效**：国际形势/议题状态变化快；`06` 的共用假设标注"截至成文（2026-06）"，并由复盘机制驱动更新。
- **越界风险**：避免滑向政治预测或投资建议；措辞条件式、聚焦劳动者的行业/职业准备，不预测政局、不荐资产。
