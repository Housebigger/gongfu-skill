# 16 集群认知框架

> 给每个行业配上"该怎么想问题"的思想武器。

## 这是什么

每个集群一份认知框架文件，把伟人思想方法论跟具体行业挂钩。引擎路由到某个集群时，不仅给行业信号（增/转/缩），还给这个行业的劳动者一套认知框架——怎么看局势、怎么判断攻守、怎么避开认知陷阱。

## 文件结构（每个集群统一格式）

1. **核心矛盾** —— 这个行业当前的主要矛盾是什么（矛盾论视角）
2. **阶段判断** —— 增量期/转型期/收缩期，该用持久战思维还是速决战思维
3. **角色分化** —— 同一行业里谁在增值、谁在被替代，分界线在哪
4. **攻守判断** —— 什么情况下该进攻（加码投入），什么情况下该防守（准备后路）
5. **认知陷阱** —— 这一行的人最容易掉进去的 3 个思维误区
6. **思想武器** —— 从方法论库里精选最相关的思想工具和篇目
