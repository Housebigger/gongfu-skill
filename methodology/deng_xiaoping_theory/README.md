# deng_xiaoping_theory/ — 邓小平理论

> "实践是检验真理的唯一标准。"
> "发展才是硬道理。"

## 概述

邓小平理论是马克思主义中国化的第二个伟大理论成果。如果说毛泽东思想解决了"革命怎么搞"的问题，邓小平理论解决的是"建设怎么搞"的问题。对今天普通劳动者来说，邓小平最有穿透力的思想不是高深的理论——而是几个极其务实、极其清醒的论断：解放思想、实事求是、发展才是硬道理、科学技术是第一生产力、让一部分人先富起来。

邓小平说话极其简练直接——没有长篇大论，每一句话都直接指向行动。这种风格跟共富参谋的服务对象（一线劳动者）天然契合。

## 与毛泽东思想、马克思主义的关系

| 思想体系 | 核心问题 | 对劳动者的价值 |
|---|---|---|
| 马克思主义 | 看清世界的底层逻辑 | 剩余价值、异化、经济基础 |
| 毛泽东思想 | 知道怎么行动 | 矛盾分析、持久战、集中兵力 |
| 邓小平理论 | 怎么务实地发展和建设 | 解放思想、实事求是、发展优先 |

三者互补：马克思帮你"看懂为什么"，毛泽东帮你"知道怎么打"，邓小平帮你"务实地干"。
