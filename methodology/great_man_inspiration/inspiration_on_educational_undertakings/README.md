# inspiration_on_educational_undertakings — 教育从业者的启发

本主题面向**教育从业者**——特别是在体制内学校（如初中）工作的教师、班主任、政治课教师——从毛泽东文选的方法论中，提取对当代教育场景真正实用的启发。

## 定位说明

教育从业者在学校里遇到的棘手问题，往往不是纯粹的"教学技巧"问题，而是：
- 如何理解和引导复杂背景的学生
- 如何在学校组织体系中有效协作、争取资源
- 如何面对危机事件（校园冲突、家校矛盾、舆情）
- 如何设计课堂让学生真正参与而非被动听讲
- 如何在繁重体制压力下保持自身成长和职业生命力

毛主席的很多方法——调查研究、群众路线、统一战线、批评与自我批评、集中优势兵力——在教育场景中有非常具体的对应。

## 子主题结构

| 子文件夹 | 定位 | 典型场景 |
|---------|------|---------|
| `indexes/` | 索引与总览 | 系列索引、阶段总结 |
| `student_understanding_and_guidance/` | 学生理解与引导 | 读懂学生真实状态、分类施策、转化后进生、调动积极性 |
| `classroom_and_teaching_design/` | 课堂与教学设计 | 课堂参与度、教学方法、让学生从"被教"变"主动学" |
| `teacher_growth_and_backbone/` | 教师成长与骨干建设 | 教师自身能力建设、骨干教师培养、避免职业倦怠 |
| `school_coordination_and_organization/` | 校内协调与组织 | 与同事协作、争取行政支持、年级组/教研组协调 |
| `school_governance_and_crisis_response/` | 学校治理与危机应对 | 校园危机处理、家校矛盾、舆情应对、制度建设 |

## 边界

- 本主题聚焦**教育从业者的工作方法论**，不是学术教育学理论。
- 侧重从毛主席文选中提取**可操作的方法和思维方式**，而非政治说教。
- 所有笔记都注意**不机械照搬战争年代语境**，而是转译为当代学校场景。
