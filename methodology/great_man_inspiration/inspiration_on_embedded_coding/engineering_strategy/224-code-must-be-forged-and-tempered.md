# 从《打退资产阶级右派的进攻》看嵌入式开发：锻就是锤打炼就是在高炉里炼——代码要经过真锤真炼

- 日期：2026-06-13
- 原文对应：《打退资产阶级右派的进攻》（1957 年 7 月 9 日）
- 原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19570709.htm
- 说明：映射到代码要经过真锤真炼。
- 核心思想当代转译："锻者就是锤打，炼就是在高炉里头炼铁，平炉里头炼钢。钢炼出来要锻。现在锻要拿汽锤锻。那个锻可厉害哩！"嵌入式代码也一样——代码不是写完就完了，要经过真锤真炼。真锤就是压力测试、边界测试、环境测试。只有经过真锤的代码才是钢，没经过的就是铁——一压就弯。

---

## 一、核心要点

- 锻就是锤打→ 压力测试
- 炼就是炼铁炼钢→ 极端环境测试
- 汽锤锻可厉害→ 真正的压力
- 没经过锤炼的是铁→ 一压就弯

## 二、给今天每个"代码写完就上线"的团队
你的代码经过真锤真炼了吗？
