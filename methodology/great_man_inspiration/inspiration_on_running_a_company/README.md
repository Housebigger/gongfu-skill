# inspiration_on_running_a_company

这个主题把毛主席相关文章转译成面向公司经营、组织设计、领导力与战略执行的方法论。

它关注的是：
- 公司在复杂局势下如何识别主矛盾
- 如何组织联盟、主战场与主力
- 如何把战略、组织、现金、节奏与人才连成一个整体

## 子目录速览

### `organization_design/`
适合看：
- 角色分层
- 组织边界
- 反馈机制
- 战略如何被组织化承接
- 管理判断如何建立在下情而不是二手汇报之上
- 如何把公司学习从概念收集改成现实研究、历史整理与方法落地
- 如何同时整顿判断作风、部门关系与管理表达，压住主观主义、山头主义与管理八股
- 如何反对公司汇报、公文、评审和会议里的空话、黑话、压人话与无对象表达

### `leadership_and_talent/`
适合看：
- 如何赢得跟随
- 如何辨认真骨干
- 如何吸纳与使用人才

### `strategy_and_execution/`
适合看：
- 公司主战场与阶段判断
- 危机治理
- 统一战线与协作边界
- 如何把暂时缓和转成结构性转机
- 如何识别假缓和、谣言离间与软性诱导，避免公司自己拆掉主线
- 如何在主要威胁突然升级时围绕共同主敌重组公司联盟、外援层与主战场
- 如何让产品、品牌、销售与服务别只服务内部自我感觉，而能进入真实客户、真实市场语言与真实一线场景
- 如何在困难阶段主动缩编公司机器、砍掉虚胖组织面并把核心业务主力做硬
- 如何在企业曙光已现时继续把人、钱、组织注意力压向主经营战场，避免最后阶段重新摊薄
