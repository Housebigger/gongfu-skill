# 从《中共中央关于九月会议的通知》看公司年度复盘：用数据复盘、设定量化目标、为下一年做好准备

日期：2026-06-13

原文对应：
《中共中央关于九月会议的通知》
原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19481010.htm

写作背景：
1948年10月10日，毛泽东为中共中央起草了《中共中央关于九月会议的通知》，向全党通报1948年9月中央政治局会议（即"九月会议"）的主要决议。这篇通知不是一份泛泛的政治宣言，而是一份极其硬核的阶段复盘加下一阶段动员文件。毛泽东在通知中做了三件事：第一，用具体数据总结过去两年的战果——歼敌多少个旅、解放多少座城市、土地改革推进到什么程度、军队扩充到多少万人，全部用事实说话，没有一句空洞的"形势大好"。第二，根据这些事实，对当前阶段做出明确判断——解放战争已经到了"从根本上打倒国民党反动派"的转折点，但距离最终胜利还需要一定时间，不能因为形势好就松懈，也不能因为还要打就悲观。第三，给下一阶段设定了具体的量化目标和行动要求——再歼敌多少个师、再解放多少个地区、军队要怎么整编、财政经济要怎么统一、干部要怎么准备。整篇通知的结构就是：用数据复盘上一阶段 → 基于事实判断当前阶段 → 设定量化目标并部署下一阶段。这个结构，今天看起来仍然是一份标准的"年度复盘 + 下一财年OKR"模板。

说明：
把这篇文章转译到公司年度复盘中，现实感极强。很多公司做年终总结、年度复盘或战略规划时，最常见的问题不是不做复盘，而是复盘没有数据、目标没有量化、下一年的准备没有落地的具体安排。复盘变成了"大家辛苦了"的年会鼓劲，目标变成了"争取做得更好"的口号，下一年的准备变成了"我们继续努力"的表态。131提醒经营者：真正有价值的年度复盘，必须用数据说话——过去一年收入增长多少、利润率变化如何、核心业务指标是进步还是退步；基于这些数据判断公司当前的位置——是在正确的轨道上加速，还是在错误的轨道上越跑越快；然后为下一年设定明确的量化目标——收入要达到什么规模、关键业务指标要达到什么数值、组织要怎么调整。没有数据的复盘是空谈，没有量化目标的部署是空转。

核心思想的当代转译：
如果把《中共中央关于九月会议的通知》翻成今天公司年度复盘的语言，我会把它概括成一句话：年度复盘必须用具体数据回顾过去一年的经营结果，基于事实判断当前业务位置和结构性问题，然后为下一年设定明确的量化目标和资源部署——不用数据的复盘是空谈，不设量化目标的部署是空转。

## 一、第一层提醒：年度复盘的第一步不是谈感受，而是把经营数据摆出来

毛泽东在通知里做的第一件事，不是分析形势、不是鼓舞士气，而是摆数据。歼敌数字、解放城市数量、军队规模、土改进度、财政状况——全部用具体数字说话。

翻到公司年度复盘里，这句话极其值钱。因为绝大多数年度复盘会的开场是这样的：

- "过去一年大家都很拼，加班很多，辛苦了。"
- "整体来说，业绩还不错。"
- "今年算是稳住了，没有出什么大问题。"
- "我们完成了大部分年度目标，还有几个延到了明年。"

这些话的共同特点是什么？没有数据。

真正有价值的年度复盘开场应该是这样的：

- 过去一年收入3.2亿，同比增长18%，但利润率从15%降到11%，增收不增利；
- 核心产品续约率从82%升到87%，但新客获取成本从1200元涨到1900元，获客效率在恶化；
- 客户投诉量从月均180单降到140单，但大客户投诉从12单升到23单，大客户满意度在下滑；
- 团队从120人扩充到165人，人均产出从267万降到194万，组织膨胀但效率没有同步提升；
- 年初定的5个关键目标，完成了3个，延后了1个，放弃了1个，具体数据和原因是……

131提醒经营者：
年度复盘的第一步不是谈感受，而是把数据摆出来。
没有数据的复盘，本质上是在用印象代替事实。
"感觉还不错"和"数据证明还不错"，是完全不同的两件事。
数据不需要完美，但必须要有。
粗略的数据也比精准的印象可靠。

## 二、第二层提醒：数据不是用来粉饰太平的，而是用来暴露结构性问题的

毛泽东在通知里不只报喜，也报了实际问题：解放区财政困难、干部数量不足、某些地区的土改执行有偏差。这些不是用来"表态承认困难"的装饰，而是用来识别下一阶段要解决的结构性矛盾。

翻到公司年度复盘中，这层意思对应的是一个极其普遍的问题：很多公司的年度复盘，变成了"挑好看的数据讲"：

- 收入涨了，大讲特讲；利润率降了，一笔带过；
- 大客户签约了，大讲特讲；中小客户流失率上升了，归因到"市场竞争加剧"就过去了；
- 新产品上线了，大讲特讲；老产品的核心指标在下滑，不在复盘会上讨论；
- 团队扩张了，大讲特讲；人均产出下降了、管理层级变厚了、决策速度变慢了，这些组织成本不在数据里呈现。

这种"选择性呈现数据"的复盘，和不做复盘差不多——因为真正决定公司下一年走向的，往往不是那些好看的数据，而是那些被跳过的问题。

131提醒经营者：
数据的价值不在于证明"我们做得不错"，
而在于暴露"我们在哪里还有结构性问题"。
收入涨了但利润率降了，说明增长是在用利润换规模；
团队扩张了但人均产出降了，说明组织效率在恶化；
新客签约了但老客流失了，说明产品竞争力在衰退。
复盘数据的时候，最值得关注的数据，
恰恰是那些"不好看"的数据。
因为那些数据背后，藏着下一年最需要解决的问题。

## 三、第三层提醒：基于数据判断公司当前阶段——是在前进，还是只是在忙碌

毛泽东在通知里做了一件非常关键的事：基于战果数据，判断当前解放战争处于什么阶段。他的结论是：已经到了从根本上打倒国民党的转折点，但还没有到最终胜利，接下来还要继续打硬仗。

这个判断的厉害之处在于：它既不是盲目乐观（"我们快赢了"），也不是保守悲观（"还要打好久"），而是基于事实的阶段定位。

翻到公司年度复盘中，这层意思解决的是一个常见问题：公司分不清"更忙了"和"更前进了"。

很多公司复盘时的表述是：

- "我们今年做了很多事。"——做了很多事，但是不是做了对的事？
- "团队加班很多，很辛苦。"——辛苦不等于有进展，忙不等于在前进；
- "基本上完成了年度目标。"——但市场占有率变了没有？竞争格局变了没有？
- "客户反馈比去年好。"——好了多少？好在哪？核心痛点有没有被解决？

131提醒经营者：
年度复盘的核心判断，不是"我们做了多少事"，
而是"公司离目标更近了多少"。
做了20个项目但核心业务指标没变，说明这20个项目可能不是决定性的；
团队从100人扩张到150人但人均产出没提升，说明规模在膨胀但能力没有升级；
看起来很忙但关键战场上的份额没有变化，说明忙在了非核心方向上。
真正有价值的年度判断只有一个标准：
这一年结束后，公司距离战略目标是不是真的更近了。
如果答案是"是的"，数据会证明；
如果答案是"不确定"，那说明这一年的目标和衡量标准本身就有问题。

## 四、第四层提醒：下一年必须设定量化目标，不能只有方向没有刻度

毛泽东在通知里最硬的部分，是他给下一阶段设定了极其具体的量化目标：再歼敌多少个旅、再解放多少个省、军队要扩充到多少人、干部要准备多少名。每一个目标都有数字，不是"继续努力"，不是"加大力度"。

翻到公司年度复盘中，这对应的是几乎所有公司都犯过的错：复盘完了，下一年的目标写成：

- "提升产品竞争力。"——提升到什么程度？用什么指标衡量？
- "优化运营效率。"——优化到什么数值？成本降多少？人效提多少？
- "加强市场拓展。"——拓展多少？从多少到多少？哪些区域？
- "改善客户体验。"——怎么衡量？NPS到多少？投诉率降到多少？

这些没有数字的目标，和没有目标是一样的。因为没有人知道"做到了"长什么样，也没有人知道"没做到"该追谁的责。

131提醒经营者：
下一年的目标，必须是量化的。
"提升产品竞争力"不是目标，"核心功能用户满意度从3.6分升到4.2分"才是；
"优化运营效率"不是目标，"人均产出从194万提升到240万"才是；
"加强市场拓展"不是目标，"华东区域市占率从12%提升到18%"才是。
量化目标的本质，不是给团队加压力，
而是让所有人知道"做成了"的标准是什么。
没有刻度的方向，不是方向，是口号。
有刻度的方向，才能变成团队共同的行动标尺。

## 五、第五层提醒：量化目标不只是数字，更要拆到具体的行动部署和责任人

毛泽东在通知里不只是说"再歼敌一百个旅"就完了，他还部署了具体的行动：各个野战军分别负责哪个方向、军队怎么整编、财政经济怎么统一、干部怎么培训和分配、新区工作怎么开展。每一个量化目标后面，都跟着具体的行动安排和责任人。

翻到公司经营里，这层意思对应的是：很多公司定了量化目标之后，就以为工作做完了。目标写在PPT里、贴在墙上、发在全员邮件里，但没有拆解成具体的行动和责任人。

现实中的场景：

- 目标是"收入增长30%"，但没有拆到具体是哪个产品线驱动增长、哪个渠道拉新、哪个区域扩张；
- 目标是"利润率提升3个百分点"，但没有指定哪些成本项要压缩、哪些业务线要提价、哪些低效项目要砍掉；
- 目标是"客户满意度提升10%"，但没有说哪些客户反馈要优先解决、哪些服务流程要优化、谁负责跟进；
- 更常见的是：量化目标定完了，但资源分配没有变化——人还是那些人、预算还是那个预算、组织架构还是那个架构，目标变成了挂在墙上的数字。

131提醒经营者：
量化目标只是起点，不是终点。
目标定了之后，必须回答三个问题：
第一，这个目标要通过哪些具体行动来实现？
第二，每个行动由谁负责、什么时间完成？
第三，现有的资源分配需不需要调整来支撑这些行动？
如果目标定了但行动没变、资源没动、人不调整，
那这个目标就不是用来执行的，而是用来汇报的。
真正有效的年度部署，是目标 + 行动 + 责任人 + 资源调整四位一体。

## 六、第六层提醒：年度复盘不只是向后看，更是为下一年做好准备——团队状态、能力储备和组织调整

毛泽东在通知里有一大段内容不是谈军事战果，而是谈组织准备：干部要准备多少、怎么训练、怎么分配；军队要怎么整编、怎么提升正规化水平；新解放区的接收和管理要怎么准备。他非常清楚：下一阶段的仗能不能打赢，不只取决于上一阶段的战果，还取决于组织有没有为下一阶段做好准备。

翻到公司年度复盘里，这层意思是最容易被忽视的：

大多数公司的年度复盘，只关注"过去一年做了什么"和"下一年做什么"。但很少有人问：

- 团队的整体状态怎么样？连续高强度运转一年后，团队是士气高昂还是已经疲态尽显？核心人才的流失风险有多大？
- 团队的能力储备够不够？下一年如果有新的业务方向或新的竞争态势，现有的能力能不能支撑？
- 组织结构需不需要调整？上一年的组织是为上一年的问题设计的，下一年的问题可能需要不同的组织配置。
- 管理层梯队建设得怎么样？关键岗位有没有backup？中层管理者的成长速度跟不跟得上公司扩张的速度？
- 制度和流程需不需要升级？上一年暴露出的效率瓶颈——审批慢、跨部门协作难、信息传递失真——下一年能不能解决？

131提醒经营者：
年度复盘的价值，不只是总结过去，
更是为下一年做好准备。
业务要准备，团队也要准备。
战略要准备，组织也要准备。
上一年的胜利，不自动带来下一年的胜利。
如果团队状态透支了、能力没跟上、组织没调整、制度没升级，
那下一年开始的时候，上一年积累的优势就会被消耗掉。
真正成熟的公司，在年度复盘的时候会花同样的精力
回答两个问题：上一年做对了什么？下一年我们准备好了吗？

## 结语：131给公司经营最硬的提醒，是用数据复盘、设定量化目标、为下一年做好全面准备

所以，131对今天公司经营最现实的提醒是：年度复盘不能只谈感受，必须用数据说话。数据不是为了粉饰太平，而是为了暴露结构性问题。基于数据判断公司当前的位置——是在正确的轨道上加速，还是在错误的轨道上越跑越快。下一年必须设定量化目标，不能只有方向没有刻度。量化目标要拆到具体行动、责任人和资源调整，不能只挂在墙上。年度复盘不只是向后看，更是为下一年做好全面准备——团队状态、能力储备、组织调整、制度升级，每一项都要检查。不用数据的复盘是空谈，不设量化目标的部署是空转，不做准备就开始的下一年是用上一年的积累为下一年的失误买单。真正有效的年度复盘，是用事实校验信心，用数据识别问题，用量化目标锁定方向，用行动部署和资源调整确保执行，最后用团队和组织的全面准备为下一年的胜利打底。
