# 从《在中国共产党全国代表会议上的讲话》看软件开发：钻进去成为内行——别浮在皮面上

- 日期：2026-06-13
- 原文对应：《在中国共产党全国代表会议上的讲话》（1955 年 3 月）
- 原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-195503.htm
- 说明：映射到技术深度。
- 核心思想当代转译："像医生一样，有的能够开刀，有的不行。有的打针能够打静脉，有的就不能，只能打皮下。有一些医生连皮下都不敢动手，就在那个皮上面。"程序员也一样——有的能深入系统底层（开刀），有的只会调用 API（打皮下），有的连 API 都不熟（浮在皮面上）。必须钻进去成为内行。

---

## 一、核心要点

- 有深有浅→ 你的技术到哪一层？
- 钻进去成为内行→ 不浮在皮面上
- 开刀>静脉>皮下>皮面→ 技术深度分级
- 新时代新问题→ 必须持续学习

## 二、给今天每个"只会调 API 不懂底层"的程序员

你在哪一层？皮面？皮下？静脉？开刀？往深处钻。
