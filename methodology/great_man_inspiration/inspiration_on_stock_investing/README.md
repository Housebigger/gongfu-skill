# inspiration_on_stock_investing

这个主题专门存放股票投资与二级市场方法论。

它关注的是：
- 市场结构与阶段判断
- 公司研究与 thesis 验证
- 风险控制、仓位与现金管理
- 交易执行与主战场识别

不处理：
- 副业项目
- 主动收入系统
- 技能变现与服务变现

那些内容属于 `../inspiration_on_making_money/`。

## 子目录速览

### `market_structure_and_cycles/`
适合看：
- 市场结构、政策周期、真实修复与虚假亢奋的区分

### `research_and_thesis/`
适合看：
- 公司研究
- thesis 搭建与验证
- 如何识别真假转折、真假盟友、真假稳定
- 如何识别管理层口中的“准备改革”“以后解决”究竟是在给 thesis 恶化、拖字维持还是真实修复打掩护
- thesis 如何避免建立在二手纪要、传闻和层层转述上
- 投资学习如何从概念收藏改成研究现状、研究历史和方法落地
- 如何识别假缓和、传闻、甩锅叙事和离间，不让软压力拆掉 thesis 主线
- 如何避免研究公共事务重新滑回少数持仓者和核心圈包办
- 如何整顿研究作风里的主观主义、小圈子偏见和研究八股
- 如何防止纪要、报告、持仓会和 thesis 复盘滑向研究八股
- 如何避免 thesis 只服务持仓立场和圈内黑话，转而进入真实需求、真实产业语言和真实验证基础
- 如何在困难阶段别只让你自己和少数研究骨干散着补 thesis，而要把分散的信息、验证面、半可用时间和研究经验组织成真正研究主力
- 如何在 thesis 责任升级时，不再只靠旧经验和少数骨干硬扛，而要研究现实、补上历史、放下包袱并训练研究机器
- 如何别把管理层信心表演误认为 thesis 修复，而要看关键回答、真实动作与风险是否真正变化
- 如何研究困难区域里的公司是否能在约束下真实生产，而不是简单否定或盲目买困境故事
