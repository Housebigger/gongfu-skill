## 子目录速览

### `survival_strategy/`
适合看：
- 如何保基本盘
- 如何建根据地
- 如何在压力下留出生存与反攻空间

### `judgement_and_decision/`
适合看：
- 识势、识人、识友方与识主矛盾
- 如何避免被表面动作和短期缓和带偏
- 如何识破假缓和、软性诱导、谣言与离间

### `long_term_growth/`
适合看：
- 长期成长、阶段推进、理解的反复锻打
- 如何把学习从背结论改成研究现实、研究历史、把理论变成会用的方法
- 如何在成长卡顿时定期整顿自己的学习、判断与表达作风，压住主观主义、小圈子成见与空话习气

### `cooperation_and_organization/`
适合看：
- 如何经营合作
- 如何组织支持面
- 如何在统一、斗争、边界、后果结构之间拿捏分寸
