## 3. 后续推进时的最小核验流程

每次准备新增一篇原文转译前，至少做这 6 步：

1. 从正式 README / overview 确认当前 DONE / NEXT。
2. 在本文件查 `NEXT` 的防错序号、标题、官方分卷和链接。
3. 重新抓取或抽查 marxists.org `index.htm` 中相邻三篇，确认前后关系没有被官网目录或本地索引改坏。
4. 搜索全仓是否已存在该编号主题文件，避免重复写。
5. 写入主题文件后，再更新各主题 README / index、总 README、sequential overview。
6. 最后核查：本文件中的下一项没有被跳过；正式索引里的当前项没有被错写成分卷内序号。
