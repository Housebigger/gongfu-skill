# 从《中国人民解放军布告》看赚钱：新项目的约法八章——开干前先立规矩

- 日期：2026-06-13
- 原文对应：《中国人民解放军布告》（1949 年 4 月 25 日）
- 原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19490425.htm
- 写作背景：渡江后毛泽东发约法八章，从打仗切换到治理。八条规矩让所有人知道新秩序怎么运行。
- 说明：本文将"约法八章"映射到新赚钱项目启动时的规则建立。
- 核心思想当代转译：开始一个新项目之前，先定好规矩——投入上限、止损点、时间限制、收益目标。不是干着看，是先想清楚再干。

---

## 一、新项目的"约法八章"

1. 最多投入 X 元 / Y 小时
2. 3 个月不盈利就砍掉
3. 不追加超过初始投入的 50%
4. 每周复盘一次数据
5. 不借钱做项目
6. 不影响主业收入
7. 合作方必须签书面协议
8. 达到目标收益后要么扩大要么退出，不原地踏步

## 二、保护 vs 砍掉——项目的区分对待

- 保护：有数据验证的项目继续投入
- 砍掉：凭感觉做、没有数据支撑的方向及时止损

## 三、给今天每个准备启动副业的人

先写好你的八条规矩。动了感情的时候，规矩帮你做决定。
