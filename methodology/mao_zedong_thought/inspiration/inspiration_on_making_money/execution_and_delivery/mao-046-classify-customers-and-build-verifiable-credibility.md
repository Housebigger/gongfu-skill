# 046《团结一切抗日力量，反对反共顽固派》对副业赚钱的启发：分层对待客户平台，用"十没有"建自己的可信度

原文对应：
《团结一切抗日力量，反对反共顽固派》

## 原文出处
- 毛泽东《团结一切抗日力量，反对反共顽固派》，1940 年 2 月 1 日
- 原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19400201.htm

## 原文核心观点（为副业从业者提炼）
1. 对不同类别的客户/平台/合作方应采取**不同的策略**，不要扩大化。
2. **两条政策并行**：拉黑/防备不良合作方 + 主动经营优质合作方。
3. 用**可验证的具体成果清单**（"十没有/十有"）来立口碑。
4. 真合作 vs 假合作：以消灭你的独立性为代价的"整合"是假合作。

## 转译到副业/搞钱的启发

### 启发一：客户分层，不扩大化

副业最常见的"扩大化"错误：
- 被一个甲方坑 → "客户都不是东西"
- 遇到一次退款 → "用户都喜欢薅羊毛"
- 被一个平台扣量 → "平台都黑"
- 被一个同行抄袭 → "同行都是敌人"
- 被一个合作方跑路 → "不和任何人合作了"

**扩大化的代价**：你变得心态封闭，新机会全部错过。

**毛的方法论翻译**：
对具体坏客户——**坚决拉黑** + 写进黑名单 + 内部同步
对客户群体——**保持分层经营**：
- A 级客户（尊重你、价格合理、愿意长期）→ 重点投入，优先排期
- B 级客户（付款偶尔拖延但没大问题）→ 正常对待
- C 级客户（反复压价、改需求、拖款）→ 提价或清退
- D 级客户（明显有坑）→ 直接不接

**一句话**：别让几个坏客户的存在，让你关闭了所有新客户的大门。

### 启发二：两条政策并行——拉黑坏人 + 主动经营好人

毛："一方面反对……一方面团结……"

副业里最常见的**单腿操作**：
- 被坑一次后整天防备心爆棚 → 老客户都觉得你怎么变刺猬了
- 一味友好不敢拒绝 → 坑你的客户继续坑

正确做法**两条政策并行**：

**"坚决防备"政策**：
- 黑名单机制：明显坑人的客户/同行直接拉黑
- 合同条款收紧：对新客户强制定金 / 改动次数 / 延期罚则
- 不惯着：遇到过分要求直接拒绝，不怕翻脸
- 保留证据：所有沟通书面化

**"积极经营"政策**：
- 老客户定期问候/节日关怀
- 主动推荐客户可能需要的新服务
- 同行建立联盟（互相引流/转介）
- 建立口碑的公开渠道（案例集/客户证言/作品集）

**两条腿同时走**，被坑一次也不会让你变成刺猬，
老客户留得住，新客户愿意来。

### 启发三：用"十没有/十有"建立自己的可信度

毛的"边区十没有"翻译到副业最直接——
**建立口碑的最好方法不是自夸，是列出可验证的事实**。

副业从业者常犯的毛病：自我介绍全是形容词——
- "我们服务好" ← 空话
- "我们性价比高" ← 空话
- "我们专业" ← 空话

**学毛的做法——写"十有/十没有"自我介绍**：

**副业/个人品牌的事实清单范例**（假设做设计）：
1. 过去 3 年完成 X 个项目，[链接到作品集]
2. 客户复购率 80%+，[可出示的合作记录]
3. 交付准时率 100%，[可查的项目台账]
4. 首稿满意率 >85%，[反馈调研数据]
5. 平均响应时间 <2 小时（工作时间）
6. 提供 3 轮免费修改，[写在合同里]
7. 无拖延交付记录，[老客户可证]
8. 支持全流程源文件交付
9. 售后 30 天答疑
10. 签订保密协议

**每一条都可验证**，不靠自夸。

反过来——**你在挑合作方/客户时也用这个标准**：
- 对方给你发"我们服务好"——抽象，别信
- 对方能给你 5-10 条可验证事实——可信

### 启发四：识别"假合作"——以消灭你的独立性为代价

副业里最常见的"假合作"形态：

- **"我们长期合作吧，你别接其他单子"** → 独占式合作，削你议价权
- **"你加入我们公司吧，挂靠名头对你好"** → 吞并你的个人品牌
- **"这个 IP 一起做，收益 5:5"** → 你投入全部精力，对方挂个名就分半
- **"把你的客户资源给我吧，一起服务更好"** → 你交出护城河

**毛的三问**：
1. 是否有真正的共同目标？（还是对方只想借用你）
2. 是否互相尊重独立性？（还是对方想消化你）
3. 是否双方都在成长？（还是你被榨干）

**典型的"真合作"特征**：
- 双方贡献清晰可量化
- 有退出机制
- 各自保留核心资产（客户/作品/品牌）
- 合同白纸黑字

**典型的"假合作"特征**：
- 对方只谈"品牌/资源/机会"，不谈分钱机制
- 要求你独占不接其他
- 要求你交出核心资产
- 口头承诺多，书面约定少

**一句话**：
以"消灭你的独立性"为代价的合作，不是合作，是吞并。
对方说得越动听，越要警惕。

## 给自己的三个自检问题
1. 我最近一次"对整个客户群体/平台/同行下整体负面判断"，是基于多少个具体样本？是否扩大化了？
2. 我现在有没有"十条可验证的事实清单"来介绍自己？如果被问"凭什么选你"，我能答出几条？
3. 我正在谈的一个合作，剥掉对方的动听话语后，看分钱机制/独立性/退出机制，是真合作还是假合作？
