# 从《党内团结的辩证方法》看软件开发：一曰斗二曰帮——对犯错的同事既要指出又要帮助

- 日期：2026-06-13
- 原文对应：《党内团结的辩证方法》（1957 年 11 月 18 日）
- 原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19571118.htm
- 说明：映射到指出错误也要帮助。
- 核心思想当代转译："对犯错误的同志第一是要斗争，要把错误思想彻底肃清。第二，还要帮助他。一曰斗，二曰帮。从善意出发帮助他改正错误，使他有一条出路。"Code Review也一样——发现同事的代码问题（斗），但不能一棍子打死——要从善意出发帮助他改正（帮）。只批评不帮助，或者只帮助不指出问题——都不对。

---

## 一、核心要点

- 斗→ 指出问题
- 帮→ 帮助改正
- 善意出发→ 不是打击
- 给一条出路→ 有改正空间
