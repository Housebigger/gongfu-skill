# 从《抗日根据地的政权问题》看投资研究治理：很多投资者不是没有体系，而是研究公共事务仍被少数持仓者和强势叙事包办，真正承担风险的人、中间证据和可合作视角都没有真实位置，最后所谓研究共治只是换了说法的旧控制

日期：2026-04-26

原文对应：
《抗日根据地的政权问题》

写作背景：
1940年3月6日，毛泽东为中共中央起草《抗日根据地的政权问题》。这篇文章表面上讨论的是抗日根据地怎样建立统一战线政权，实际上处理的是一个非常具体而现实的治理难题：当一个系统既要坚持主方向，又要把更多可合作的人组织进来时，结构究竟该怎样设计，领导地位究竟该怎样实现，合作面究竟该怎样扩大，才不会一边喊共同治理，一边又滑回少数人包办、盛气凌人、把公共事务重新私有化的老路。文中提出了极有方法感的几层要求：政权必须是统一战线性质的；人员结构上要有明确的三三制安排；领导不能靠口号和压服，而要靠正确政策与模范工作去说服他人；要克服不愿与党外人士合作的狭隘性；遇事先商量并取得多数同意；鼓励党外人士对各种问题提出意见并认真倾听；不能因为自己手里有军队和政权，就要求一切无条件照自己的决定去做。

说明：
今天把这篇文章转译到股票投资语境里，当然不能机械照搬战争年代关于根据地、政权和阶级联合的政治表达。但它背后的那套治理方法，对今天任何认真做研究、做仓位、做复盘、做信息整合的人都特别有用。很多投资者并不是没有框架，也不是没有研究流程，更不是没有群、没有会、没有资料，而是研究公共事务仍被少数持仓者、少数强势叙事、少数历史收益最好的人把持着。真正承担风险的人没有真实位置，中间证据没有被认真争取，可合作的异见视角被当噪音而不是伙伴，最后所谓投研共识就变成一种更体面的包办。表面上有讨论，实际上还是旧控制；表面上说团队研究，实际上真正能定框架、能改判断、能改变仓位结构的人还是那几个。

这套问题在投资里尤其隐蔽，因为“谁有话语权”常常不是靠正式头衔体现，而是靠收益神话、历史战绩、仓位大小、资源密度、信息先手和风格地位来体现。久而久之，一个本来应该越来越成熟的研究系统，很容易慢慢变成：大仓位的人说了算、强叙事的人说了算、过去赢过的人说了算、最会表达的人说了算，而真正承担回撤的人、真正能提供中间证据的人、真正能拉回验证的人却越来越边缘。结果不是研究更强，而是系统越来越像“带着研究外衣的小圈子控制”。

核心思想的当代转译：
如果把《抗日根据地的政权问题》翻成今天投资研究的语言，我会把它概括成一句话：
很多投资者不是没有体系，而是研究公共事务仍被少数持仓者和强势叙事包办，真正承担风险的人、中间证据和可合作视角都没有真实位置，最后所谓研究共治只是换了说法的旧控制。真正稳的投研治理升级，不是让少数人继续垄断判断，也不是让所有人无边界抢结论，而是把最该参与的人组织进来，让结构有层次、表达有入口、异见有位置、修正有机制；而真正成熟的主导，不靠“我仓位更大”“我以前更准”“我资源更多”来压人，而靠框架更对、证据更硬、复盘更扎实、愿意商量并能把别人说服进来。

## 一、第一层提醒：很多投研系统不是没有流程，而是研究公共事务一直还被少数人包办

这篇文章最现实的一点，
是它并不把“有政权”自动等同于“治理已经对了”。

翻到今天投资研究里，
这层提醒特别值钱：
很多研究系统真正的问题，
并不是完全没有框架，
而是框架背后的研究公共事务，
一直还是少数人在包办。

现实里很常见的样子是：
- 名义上大家都能讨论，实际上结论总被少数人提前定掉
- 表面上谁都可以提反对意见，实际上真正能改仓位的人一直就那几个
- 看起来是开放研究，实际上还是强者叙事治理
- 名义上是共识形成，实际上关键判断永远在小范围里被关门处理
- 真正承担回撤后果和组合波动的人永远在执行层，进不了形成层

这种局最容易制造一种错觉：
好像系统已经“机构化”“流程化”了。

但如果研究公共事务的真实归属没有变，
那它本质上还是旧秩序。

也就是说，
不是真没研究，
而是研究始终没有公共化。

## 二、第二层提醒：真正成熟的投研治理，不是只有最强叙事者有位置，而是把承担者、进步力量和中间证据都组织进来

文章里提出三三制，
最值得今天投资者学的，
并不是数字本身，
而是一种结构思维：
一个研究系统如果想稳，
就不能只靠“最像自己的人”把全部位置占满。

今天很多投资者最容易犯的错就是：
- 只信最像自己风格的人
- 只用最支持自己结论的人
- 只给最会讲故事的人位置
- 只让最先站队的人参与核心判断

这样做短期看似高效，
长期却很危险。

因为一个投研系统真正要稳，
往往要同时满足三类力量都有真实位置：
- 真正承担组合后果和风险预算的人
- 真愿意推动研究升级、把问题做实的人
- 不极端、可合作、能稳定系统的中间证据与桥梁视角

如果一个系统只有第一类人，
很容易沦为仓位意志治理；
如果只有第二类人，
容易冲得快但站不稳；
如果完全没有第三类人，
研究面很难真正做宽。

放到投资现实里，
这种应该被组织进来的力量往往包括：
- 真正承担回撤和波动的人
- 做产业验证、财报跟踪、估值比较的人
- 能提供中性反证的人
- 不完全站队但能稳定讨论质量的人
- 帮你把结论拉回证据的人

所以第二层提醒是：
真正成熟的投研治理，
不是把所有位置都塞给“自己人”，
而是让不同角色、不同倾向、不同风格但可合作的人，
都在结构里找到真实位置。

## 三、第三层提醒：主导地位不能靠仓位、资源和历史战绩在手来维持，而要靠框架更对、证据更硬和持续说服

文章里有一句特别值得今天投资者反复想：
所谓领导权，
不是高喊口号，
也不是盛气凌人地要人家服从，
而是以正确政策和模范工作，
说服和教育党外人士，
使他们愿意接受建议。

翻到今天投资研究里，
这几乎可以直接变成一条硬标准：
真正成熟的主导，
不是因为你仓位更大、收益更高、资源更多，
别人就必须照你的结论做。

很多研究系统里，
最大的问题不是没有核心主导者，
而是主导者对“领导”两个字的理解太旧：
- 觉得我资金量更大，所以意见更该优先
- 觉得我历史收益更好，所以不同意见就是不懂市场
- 觉得我信息更多，所以结论不必解释
- 觉得我认识的管理层更多，所以验证不必公开

这种主导方式看起来很有控制感，
其实特别不稳。

因为别人真正服的，
从来不是你先占了话语位置，
而是：
- 你框架是不是更对
- 你证据是不是更硬
- 你归因是不是更清楚
- 你复盘是不是更扎实
- 你是不是愿意把异见真正拉进来再判断

真正成熟的研究主导，
不是把音量做大，
而是把可信度做高。

你不必把所有研究入口都抓在手里，
也照样可以形成主导；
但如果你只会抓入口，
却做不到框架更对、证据更硬、解释更清楚，
那你占再多位置，
别人也只是被迫附和，
不会真正形成共治。

## 四、第四层提醒：不要把中间证据和中间视角都当成麻烦，很多投研系统坏就坏在永远不会认真争取中间力量

文章里反复强调，
必须争取中间派。

翻到今天投资研究里，
这层提醒特别现实：
很多系统并不是倒在明确错误结论手里，
而是倒在一种很幼稚的研究习惯里——
只会区分“完全看多我”和“完全反对我”，
不会认真争取那些本来可以稳定判断质量的中间力量。

投资里的中间力量是什么？
往往是：
- 不完全支持你，但也不敌对的产业证据
- 不够热烈，却更接近真实的中性数据
- 不一定站在你这边，但愿意把问题讲清的人
- 不负责最终拍板，却能让结论少走偏的人
- 不追随强叙事，却能稳定研究气氛的人

很多投资者一遇到复杂局面，
就很容易陷入一种幼稚姿态：
- 不是完全认同我的就不值得听
- 不够热情的就是没价值
- 中性观点都是骑墙
- 提醒风险的人是在拖后腿

这样做的结果往往是：
核心叙事越收越紧，
支持面越做越窄，
一旦市场承压，
就没有缓冲层、没有桥梁层、没有稳定层。

所以第四层提醒是：
成熟的投资研究者，
不只会保护核心判断，
也会认真经营中间证据。

## 五、第五层提醒：真正的研究合作，不是把别人拉来站队，而是让别人能表达、能商量、能影响判断过程

文章里有一段特别有现实感：
要提倡民主作风，
遇事先商量，
取得多数同意，
同时鼓励党外人士提出意见，
并认真倾听这些意见。

这段话今天读起来，
像是在直接敲很多投研系统的毛病。

很多人嘴上也说团队研究、开放讨论，
但他们心里的合作其实是：
- 我先定结论，你来补充材料
- 我通知你观点，你去配合验证
- 我征求一下意见，但并不准备改
- 我让你发言，但不会让你影响仓位过程

这不叫合作，
只是更礼貌的单向控制。

真正的研究合作至少要满足几件事：
- 对方能提前进入，而不是结论定完才被通知
- 对方能提出真实不同意见，而不是只能补充细节
- 对方的意见会影响判断过程，而不是只做装饰
- 多数同意是被认真争取的，而不是形式上的点头

如果一个投研系统从头到尾都不愿意先商量，
不愿意真听，
不愿意让别人影响判断过程，
那它再怎么强调共识研究，
也很难真正走出包办逻辑。

## 六、第六层提醒：别把“我手里有资金、信息和历史战绩”误当成不必解释、不必协商的理由

文章里有一句非常值得今天反复记住的话：
绝不能以为我们有军队和政权在手，
一切都要无条件地照我们的决定去做。

放到今天投资研究里，
它背后说的是一种特别常见的堕落：
一旦手里有资源，
就开始觉得自己不必解释、不必协商、不必说服。

现实里的资源可以是很多东西：
- 钱在我手里
- 仓位是我拍的
- 管理层资源是我搭的
- 信息渠道我更多
- 历史收益我最好
- 社群号召力我最强

这些当然都会带来更大影响力。

但它们绝不是：
“所以别人该无条件照我的决定做”的理由。

因为研究一旦滑到这一步，
系统就会很快变成：
- 强者不必解释
- 异见不必认真处理
- 中间证据不必被纳入
- 错误归因不必公开修正
- 资源拥有者逐步把研究公共事务私有化

最后留下来的，
就不是成熟主导，
而是旧控制。

## 七、第七层提醒：真正成熟的投资研究共治，不是压成一种声音，而是在差异中建立稳定合作

这篇文章给今天投资者最大的启发，
不是一句抽象的“要共识”，
而是：
真正稳的共治结构，
不是靠把差异压没，
而是靠在差异中组织合作。

放到今天研究系统里，
真正成熟的共治应该是什么？

不是：
- 只有一种声音
- 只有一个研究框架可以说话
- 只有最大仓位的人配拍板
- 只有强叙事能定义现实
- 只有老成员才有形成层位置

而是：
- 承担风险的人有位置
- 中间证据有位置
- 反证和异见有入口
- 真正做事和真做验证的人能影响过程
- 主导者靠正确、过硬、能说服形成领导

真正成熟的投资研究治理，
不是把所有人压成一个样，
而是在差异中建立稳定合作，
让研究系统既能形成方向，
又保留修正能力。

## 结语：别把“研究共识”做成强势叙事包办公共事务的借口

《抗日根据地的政权问题》给今天投资研究最大的提醒，是：
不要把“共识”“团队研究”“统一判断”这些好词，
慢慢做成少数持仓者和强势叙事包办研究公共事务的借口。

很多投研系统最终变旧，
并不是因为没有聪明人，
而是因为它越来越不愿把真正该进入的人组织进来，
越来越不愿认真对待中间证据和异见视角，
越来越习惯于让资源在手者直接定义现实。

真正成熟的投资研究治理，
应该做到的是：
既能形成主导，
又不把主导做成控制；
既能争取共识，
又不消灭差异；
既能提高效率，
又不把效率做成封闭系统的借口。

只有这样，
所谓研究共治才不是空壳，
而会真正变成一种不断提高判断质量、提高修正能力、提高长期胜率的治理能力。
