# Marxism / 马克思主义

> "全世界无产者，联合起来！"

## 概述

本目录存放马克思主义经典文献（原料库）及其当代转译（启发库）。

马克思主义是共富参谋思想根基的重要组成部分。毛泽东思想是马克思主义中国化的成果，而马克思主义本身——尤其是马克思对资本、劳动、剩余价值、阶级的分析——对理解当代劳动者的处境有着不可替代的穿透力。

## 与毛泽东思想的关系

- **毛泽东思想** 是马克思主义在中国具体条件下的创造性发展，侧重实践策略、组织方法、矛盾分析
- **马克思主义原典** 侧重对资本主义的结构性批判——剩余价值、劳动异化、资本积累、周期性危机
- 两者互补：马克思主义帮你"看清世界的底层逻辑"，毛泽东思想帮你"知道该怎么行动"

## 原料库规划

### 马克思（marx/）
优先收录与劳动者最相关的篇章：
- 《雇佣劳动与资本》— 劳动者为什么赚不到钱
- 《工资、价格和利润》— 工资的本质
- 《资本论》（节选）— 剩余价值、劳动异化
- 《关于费尔巴哈的提纲》— 实践观点
- 《政治经济学批判》序言 — 经济基础与上层建筑

### 恩格斯（engels/）
- 《英国工人阶级状况》— 工人阶级的真实处境
- 《反杜林论》（节选）— 社会主义从空想到科学
- 《家庭、私有制和国家的起源》（节选）

### 列宁（lenin/）— 后续阶段
- 《怎么办？》— 组织方法论
- 《帝国主义是资本主义的最高阶段》
