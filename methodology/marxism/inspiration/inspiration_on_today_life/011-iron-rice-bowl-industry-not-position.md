# 从《家庭、私有制和国家的起源》看环卫/公交司机为什么"稳定但不安全"——铁饭碗的"铁"指行业不指岗位

日期：2026-06-22

原文对应：
《家庭、私有制和国家的起源》（弗里德里希·恩格斯，1884年）

说明：
恩格斯这本巨著追溯了人类社会中"制度"的演化——家庭形态、私有制、国家机器都不是永恒的，而是在特定历史条件下产生的，也会在条件变化时改变。这个思维方法对今天公用事业（环卫/公交/供水/燃气）的劳动者特别有用：你的行业是"铁"的（永远不会消失），但你的岗位不是"铁"的——机械化替代、自动驾驶、智慧城市都在改变"岗位"的形态。恩格斯教你用演化的眼光看自己的职业。

核心思想的当代转译：
恩格斯的核心方法不是告诉你"制度会变成什么"，而是教你"怎么看制度的演化趋势"。对公用事业劳动者来说：城市永远需要清洁、供水、供气、公共交通——这是永恒的。但"怎么清洁""怎么供水""怎么交通"一直在变。你的安全感不来自"这个岗位永远在"，而来自"我能不能跟着变化走"。

## 一、恩格斯的方法：一切制度都是历史的，不是永恒的

恩格斯花了大量篇幅证明：私有制、家庭形态、国家政权都不是"天然如此"，而是在生产力发展到特定阶段的产物。

这意味着：当生产力继续发展，这些制度也会改变。

这个方法翻译到你的职业：
- "环卫工人 = 拿扫帚扫地" → 这不是永恒的。当机扫车成本低于人工，这个定义就会变。
- "公交司机 = 手握方向盘" → 这不是永恒的。当自动驾驶成熟到能跑固定线路，这个定义就会变。
- "抄表员 = 逐户抄水表" → 这已经不是了。智能水表消灭了这个岗位。
- "收费员 = 在收费站收费" → ETC 已经大面积替代了。

恩格斯会说：不要把"当前的样子"当成"永远的样子"。你的行业是稳定的，但你行业里的岗位一直在演化。

## 二、环卫工人的演化路线

恩格斯式的分析：

阶段 1（过去）：人工清扫 → 扫帚 + 人力车 → 劳动密集、收入极低
阶段 2（现在）：机扫替代 → 机扫车 + 洒水车 + 垃圾分类 → 开车的人替代扫地的人
阶段 3（未来）：智慧环卫 → 传感器监控 + AI 调度 + 自动驾驶清扫 → 会操作系统的人替代只会开车的人

每个阶段，"环卫工人"这个名字没变，但干的事、需要的技能、收入水平完全不同。

如果你还在阶段 1（人工清扫），你要做的不是"更卖力地扫"，是赶紧学机扫车操作——进入阶段 2。
如果你在阶段 2（开扫车），你要做的是学一点垃圾分类运营、设备维护——往阶段 3 准备。

## 三、公交/出租/网约车司机的时间窗口

恩格斯的方法告诉你：自动驾驶不是"会不会来"的问题，是"什么时候来、先来哪一段"的问题。

演化路线：
- 第一步：高速公路自动驾驶（已落地）→ 长途货运司机先受冲击
- 第二步：固定线路公交自动驾驶（试点中）→ 公交司机 5-8 年内开始受影响
- 第三步：城市出租车/网约车自动驾驶（Robotaxi 试点中）→ 8-15 年内规模化
- 第四步：全场景自动驾驶 → 15-20 年

你不是明天就失业。但恩格斯的方法告诉你：**在"制度"还没完全变之前，你有一个窗口期来准备**。这个窗口期就是你的"转型黄金期"——现在不准备，等到变化来了再准备就来不及。

公交/出租车司机在窗口期内可以做的：
- 考一个特种运输证（危化品/大件运输）→ 自动驾驶更难替代的领域
- 学物流管理 → 从"开车"变成"管车队"
- 学车辆维修 → 从"开车"变成"修车"，维修比驾驶更难自动化

## 四、清醒的启示

1. **恩格斯的核心方法是"演化思维"**：不要用静态的眼光看自己的岗位。今天你干的活，5 年后可能由机器干。你的任务是预判趋势，提前走到机器够不着的地方。

2. **行业的"铁"给你的是缓冲时间，不是永久保障**。环卫行业不会消失，但人工清扫的岗位在减少。你有一个 5-10 年的缓冲期来转型——这个时间足够你考一个证、学一门技能。但如果你什么都不做，缓冲期结束的时候就是你被替代的时候。

3. **公用事业的优势是"稳定"——但稳定本身也是陷阱**。太稳定的环境容易让人丧失"变化的意识"。私营行业的劳动者每天都在感受竞争压力，反而更有转型动力。公用事业的劳动者因为"暂时还稳"，容易麻痹。恩格斯告诉你：越是在稳定的环境里，越要主动看趋势。

4. **最好的铁饭碗不是"某个稳定的岗位"，是"你能不断适应新岗位的能力"**。行业是铁的，岗位是活的。你的安全感应该建立在"我能跟着变"上，而不是"这个岗位不会变"上——因为岗位一定会变。
