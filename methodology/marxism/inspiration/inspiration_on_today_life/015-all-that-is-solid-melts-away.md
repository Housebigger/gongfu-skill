# 从《共产党宣言》"一切固定的东西都烟消云散了"看你为什么总在焦虑——不是你矫情，是世界真的在不停碎

日期：2026-06-22

原文对应：
《共产党宣言》（卡尔·马克思、弗里德里希·恩格斯，1848年）

> "一切固定的僵化的关系以及与之相适应的素被尊崇的观念和见解都被消除了，一切新形成的关系等不到固定下来就陈旧了。一切等级的和固定的东西都烟消云散了，一切神圣的东西都被亵渎了。人们终于不得不用冷静的眼光来看他们的生活地位、他们的相互关系。"

说明：
这是《共产党宣言》中最广为引用的一段。马克思和恩格斯在描述资本主义对传统社会的摧毁——它把所有稳定的关系、固定的身份、代代相传的手艺，全部变成了可以买卖的商品。今天这段话比任何时候都更真实：你父母的"铁饭碗"消散了、"一门手艺吃一辈子"消散了、"在一家公司干到退休"消散了。你的焦虑不是矫情——是世界真的在不停碎裂，而你被要求在碎裂中保持稳定。

核心思想的当代转译：
马克思揭示的不是"某个东西消散了"，而是"消散本身成了常态"。在传统社会，消散是异常——一个人的身份、职业、关系基本是固定的。但在资本主义社会，消散是规则——每过几年，你熟悉的一切就可能被重新洗牌。你的焦虑感就是这种"持续消散"在心理上的投影。

## 一、"消散"的清单：你失去了哪些"曾经固定的东西"

对照宣言，列出今天"烟消云散了"的东西：

职业身份的消散：
- "我是钢铁厂的工人" → 钢铁厂在收缩
- "我是教培老师" → 教培行业清零了
- "我是报社记者" → 报社在关门
- "我是网约车司机" → 自动驾驶在逼近

技能的消散：
- "我会做网页" → AI 一键生成
- "我会做设计" → AI 替代
- "我会写文案" → AI 替代
- "我会做财务报表" → 软件替代

关系的消散：
- "在一家公司干到退休" → 消散了
- "师徒制传帮带" → 消散了
- "同事是一辈子的朋友" → 消散了
- "客户关系靠人脉维持" → 平台化消散了

信仰的消散：
- "努力就能成功" → 你见过太多努力但失败的人
- "知识改变命运" → 学历在贬值
- "老实人有好报" → 你发现"老实"在职场上是劣势

马克思 178 年前就说了：这就是资本主义的本性。它不停破坏旧的结构，制造新的。它不停消解"固定的东西"，让你不得安宁。

## 二、消散的双重性：它摧毁了你依赖的，但也打开了你被锁住的

马克思是一个极其冷静的分析者。他既看到了消散的痛苦，也看到了消散的解放：

"一切等级的和固定的东西都烟消云散了" → 这意味着旧的等级制度也被打破了
"一切神圣的东西都被亵渎了" → 这意味着旧的不合理的权威也被挑战了

翻译成今天：
- "铁饭碗"消散了 → 但你也不必一辈子被绑在一个你不喜欢的地方了
- "一门手艺吃一辈子"消散了 → 但你也有机会不断学新东西、转新方向了
- "在一家公司干到退休"消散了 → 但你也不必忍受一个糟糕的老板一辈子了
- "学历决定一切"消散了 → 但没有名校背景的人也有机会了

消散摧毁了你依赖的东西，但也打开了你被锁住的东西。

问题在于：大多数人只感受到"被摧毁"的痛苦，看不到"被打开"的可能性——因为他们还在用旧框架理解新世界。

## 三、"用冷静的眼光看自己的生活地位"

宣言最后一句特别重要："人们终于不得不用冷静的眼光来看他们的生活地位、他们的相互关系。"

"冷静的眼光"——这就是马克思给你的方法。

不要怀念已经消散的东西——因为它不会回来。
不要怨恨消散本身——因为消散是规律，不是某个人的阴谋。
要用冷静的眼光看清：
1. 什么正在消散？（你的行业/岗位/技能有没有在贬值？）
2. 什么正在形成？（有没有新的需求/机会/岗位在出现？）
3. 你在消散和形成之间的哪个位置？（你是"被消散的那一方"还是"能抓住新形成的那一方"？）

## 四、清醒的启示

1. **你的焦虑有正当性**。不是你矫情、不是你脆弱——是世界确实在加速碎裂，而你被要求在碎裂中正常运转。接受这个焦虑的正当性，反而能让你不那么焦虑——因为你知道"不是我出了问题"。

2. **但焦虑不能止步于焦虑**。马克思写宣言的目的不是让你焦虑——是让你清醒。清醒之后做什么？做"能在消散中存活"的事：
   - 不要把全部身份绑在一个正在消散的岗位上
   - 不要把全部收入绑在一个正在消散的渠道上
   - 不要把全部技能绑在一个正在消散的工具上

3. **"消散"是你的敌人，也是你的盟友**。如果世界是"固定的"，你就永远不可能突破——因为你是后来者，好位置都被占了。正是因为世界在消散，后来者才有机会。你的任务不是阻止消散，是在消散中找到新的位置。

4. **用"冷静的眼光"做人生规划**。问自己：5 年后我现在的岗位还在吗？如果不在，我能做什么？如果现在就开始准备，5 年后我不但不怕消散，反而能利用消散。这就是马克思说的"冷静的眼光"——不恐惧、不逃避、不幻想，只看清事实，然后行动。
