# xi_jinping_thought/ — 习近平新时代中国特色社会主义思想

> "人民对美好生活的向往，就是我们的奋斗目标。"

## 概述

习近平新时代中国特色社会主义思想是马克思主义中国化的最新成果，是当今时代中国国家发展的指导性思想。对共富参谋来说，这套思想体系最核心的价值在于：它直接定义了"未来5-15年中国往哪走"，而这正是每一个劳动者"我该往哪走"的大坐标。

习近平思想对劳动者的直接价值：
- "以人民为中心" → 任何行业判断的终极标准：是不是在服务人民
- "新质生产力" → 未来技能投资的方向：数字化/绿色化/智能化
- "高质量发展" → 不再拼规模拼效率——个人也一样，不拼体力拼价值
- "共同富裕" → 共富参谋的名字来源和终极使命
- "人类命运共同体" → 出海/跨境/国际合作的大背景
- "绿水青山就是金山银山" → 绿色能源/环保/生态的职业方向

## 四套思想体系的关系

| 思想体系 | 核心问题 | 对劳动者的价值 |
|---|---|---|
| 马克思主义 | 看清世界的底层逻辑 | 剩余价值、异化、经济基础 |
| 毛泽东思想 | 知道怎么行动 | 矛盾分析、持久战、集中兵力 |
| 邓小平理论 | 怎么务实地发展和建设 | 解放思想、实事求是、发展优先 |
| 习近平思想 | 当今时代往哪走 | 新质生产力、高质量发展、以人民为中心、共同富裕 |

四者递进：马克思帮你看懂 → 毛泽东帮你打仗 → 邓小平帮你实干 → 习近平帮你看清方向。
