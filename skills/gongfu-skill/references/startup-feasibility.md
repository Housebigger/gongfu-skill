# 创业可行性评估 skill

## 概述

这个 skill 把 strategy/entrepreneurship/ 的四条零成本创业路径 + 诚实劝退逻辑，蒸馏成一个**可调用的创业诊断服务**。用户输入自己的技能/资源/财务/家庭状况，skill 产出一份诚实的创业可行性判断。

它解决的问题是：**「我这个人，在现在这个处境下，该不该创业？创什么？怎么起步？」**——不是鼓励你冲，也不是劝你别动，是帮你做一次清醒的自我评估。

## 什么时候用

- 想创业但不确定自己够不够格
- 已有副业但不确定要不要全职投入
- 想帮亲友评估创业想法是否靠谱
- 辞职创业前做最后一次冷静检查

**不要用于**：融资路演准备、企业战略咨询、连锁加盟决策——这些需要专业商业分析，超出了本 skill 的边界。

## 输入规格

| 参数 | 必填 | 说明 | 示例 |
|---|---|---|---|
| 你会的技能 | 是 | 能直接帮别人解决问题的本领 | 会修家电/会做菜/会剪辑视频/懂财税 |
| 财务状况 | 是 | 有多少月生活费结余 | 有6个月结余/月光/负债 |
| 家庭态度 | 是 | 家人是否支持/反对 | 支持配偶/反对的父母/单身 |
| 时间状况 | 否 | 每天有多少业余时间 | 上班族每天2小时/全职带娃 |
| 可用资源 | 否 | 车辆/设备/场地/资质/客户 | 有面包车/有电工证/有院子 |
| 所在地 | 否 | 影响可行路径 | 一线城市/县城 |

## 执行逻辑

### 第 1 步 · 劝退检查（先做这个！）

先检查 7 种不建议创业的情况（来自 entrepreneurship/07 诚实劝退）：

| 检查项 | 如果命中 | 建议 |
|---|---|---|
| 家庭财务紧张到下个月房贷还不上 | ❌ 暂不建议 | 先稳定主业，恢复到有3—6个月结余 |
| 家人强烈反对且无法沟通 | ❌ 暂不建议 | 先沟通，硬做是赌气不是创业 |
| 创业原因是「不想打工了」 | ⚠️ 重新想想 | 「逃离」不是「向往」，先想清楚走向什么 |
| 没有任何可变现技能 | ❌ 先学技能 | 技能是创业的前提，去 growth_path |
| 信「三个月月入十万」 | ❌ 你是别人的项目 | 远离割韭菜，建立现实预期 |
| 只想当老板不想干活 | ❌ 认清现实 | 个体户比打工更累 |
| 所在行业增量窗口已关闭 | ⚠️ 换赛道 | 在收缩赛道创业成功率极低 |

**如果命中任何一条 ❌，先给出劝退建议，不进入后续步骤。** 这是这个 skill 最重要的设计——不是所有评估都应该「鼓励前进」。

### 第 2 步 · 适合信号检查

检查 5 个适合创业的信号，命中 ≥3 条则条件基本具备：

1. ☐ 手里有一门能直接帮别人解决问题的技能
2. ☐ 主业稳定，有业余时间
3. ☐ 有 3—6 个月生活费结余
4. ☐ 家人至少不强烈反对
5. ☐ 看到了真实的、反复出现的、别人愿意付钱的需求

### 第 3 步 · 路径匹配

根据用户的技能/资源，匹配四条路径中最适合的一条：

| 路径 | 匹配条件 | 阶段标记 |
|---|---|---|
| 服务化 | 有能直接变现的手艺（维修/美容/做饭/设计/护理） | 从【试】开始 |
| 内容化 | 有3年+经验 + 基本表达能力 | 从【试】开始（持久战，1—3月见效） |
| 电商化 | 有选品眼光 + 基本运营能力 | 从【试】开始 |
| 技能出租 | 有可出租资源（车/设备/场地/资质） | 从【试】开始 |

**优先推荐原则**：见效最快的优先（服务化 > 技能出租 > 电商化 > 内容化）。

### 第 4 步 · 节奏建议

根据用户当前阶段，给出节奏建议：

- **【试】验证期（第1—3月）**：不辞职，每天1—2小时，投入≤500元，目标=卖出第一单
- **【稳】起步期（第4—12月）**：仍不辞职，副业有稳定小额收入，目标=系统化
- **【扩】发展期（副业超主业3个月后）**：才考虑全职投入

**铁律**：副业收入稳定超过主业工资的3个月之前，不要辞职。

### 第 5 步 · 红线提醒

无论评估结果如何，必须输出 7 条合规红线：
1. 不挂证（执业证只跟服务走）
2. 不无证经营（需资质的行业先考证）
3. 不虚假宣传（尤其食品/保健品/化妆品）
4. 不传销（收入靠卖货不靠拉人头）
5. 不刷单（假数据不带来真客户）
6. 不侵权（用自己内容、拍自己图）
7. 不碰金融揽储（零成本创业是赚客户的钱，不是拿别人的钱投资）

## 测试用例

### 用例 1：有技能但月光 + 家人反对

输入：技能=会修家电，财务=月光，家庭=妻子反对，时间=每天2小时

期望输出：
- 劝退检查：❌ 未通过（命中「财务紧张」+「家人强烈反对」）
- 建议：先稳定财务到有3个月结余，先和妻子沟通计划+止损线
- 不进入路径推荐

### 用例 2：电工 + 有余粮 + 家人支持

输入：技能=电工/持电工证，财务=有6个月结余，家庭=支持，时间=周末，资源=有面包车+电工证+社区群

期望输出：
- 劝退检查：✅通过
- 适合信号：5/5命中
- 推荐路径：服务化（技能出租也可）——持证电工上门维修
- 阶段：【试】第一周在社区群发服务信息，周末接第一单
- 红线：电工证跟人不跟钱（不挂证）

### 用例 3：什么都不会的应届生

输入：技能=无，财务=父母支持，家庭=支持，时间=全职

期望输出：
- 劝退检查：❌ 未通过（命中「没有可变现技能」）
- 建议：先去 growth_path 学一门技能（3—6个月），技能是创业前提不是结果

## 诚实的边界

1. 本 skill 的核心价值之一是**劝退**——不是所有评估都该鼓励前进。如果命中劝退条件，skill 会明确说「现在不是时机」。
2. 「零成本」指不负债/不囤货/不租铺，不是「保证赚钱」。创业永远有风险。
3. 本 skill 不提供融资/加盟/连锁建议，不替代法律/税务咨询。
4. 评估基于用户自述信息，实际可行性需用户自行验证。
