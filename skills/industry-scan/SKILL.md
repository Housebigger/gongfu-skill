# 行业机会扫描 skill

## 概述

这个 skill 把 strategy/worker_guidance/ 的 16 产业集群判断 + strategy/regional/ 的五大区域差异，蒸馏成一个**可调用的行业诊断服务**。用户输入自己的行业、岗位和所在区域，skill 产出一份个性化的行业前景判断。

它解决的问题是：**「我这个行业，在我这个地方，未来几年到底行不行？」**——不是宏观政策解读，是针对一个具体的人给出具体判断。

## 什么时候用

- 一线劳动者想判断自己所在行业的前景
- 想转行但不知道目标行业好不好
- 应届生选方向时比较不同行业
- 返乡/跨地域就业时了解地域差异

**不要用于**：投资建议、炒股决策、企业战略咨询——这些超出了本 skill 的边界。

## 输入规格

用户需要提供以下信息（最少前三项，越完整判断越准）：

| 参数 | 必填 | 说明 | 示例 |
|---|---|---|---|
| 行业 | 是 | 你所在的行业或感兴趣的行业 | 制造业/餐饮/编程/养老护理 |
| 岗位 | 是 | 你的具体岗位或想做的岗位 | 产线操作工/厨师/程序员/护理员 |
| 地域 | 是 | 你在哪个区域（五大区域之一或具体省市） | 长三角/县城/东北/西部 |
| 年龄段 | 否 | 影响转型建议 | 25岁/40岁/55岁 |
| 现状 | 否 | 在岗/待业/想转行/应届 | 在岗产线工人，干了10年 |

## 执行逻辑

### 第 1 步 · 行业归类

把用户输入的行业映射到 16 集群（A—P）之一或多个：

| 集群 | 覆盖行业 |
|---|---|
| A 先进制造与硬科技 | 集成电路/工业母机/高端装备/新材料/基础软件/生物制造 |
| B 数字与智能产业 | AI/算力/软件/数据要素/工业互联网 |
| C 绿色能源全链 | 光伏/风电/水电/核电/储能/电网/碳市场 |
| D 农业与乡村振兴 | 种植养殖/种业/农机/县域经济 |
| E 民生服务 | 养老/托育/医疗健康/创新药/家政 |
| F 文化创意与出海 | 网文/游戏/影视/微短剧/动漫 |
| G 基建物流房地产 | 交通/低空/物流/建筑/房地产 |
| H 新兴未来产业 | 低空/机器人/商业航天/氢能/量子 |
| I 传统矿业 | 煤炭/油气/金属矿/非金属矿 |
| J 传统轻纺 | 纺织服装/食品饮料/家具/玩具 |
| K 传统重化工 | 钢铁/有色/化工/水泥/玻璃 |
| L 商贸零售餐饮 | 批发零售/电商/餐饮/酒店 |
| M 金融与商务服务 | 银行/保险/证券/法律/会计/咨询 |
| N 教育与培训 | 基础教育/教培/职业教育/在线教育 |
| O 居民生活服务 | 美容美发/修理/婚庆/宠物/健身 |
| P 公用事业与市政 | 环卫/供水燃气/市政/环保/公交 |

### 第 2 步 · 信号判定

对用户所在的行业+岗位组合，判定信号：

- **【增】确定性增量**——十五五明确投入、需求真实增长、岗位在扩。建议：积极进入/加码技能。
- **【转】转型必经路**——行业不消失但必须升级。不转型会被淘汰，转了有机会。建议：主动学新技能。
- **【缩】收缩高风险**——产能过剩/政策限制/技术替代。建议：准备后路，不要恋战。

判定依据：
1. 该集群在 worker_guidance 中的总体信号
2. 该具体岗位在集群内的分工种行动建议中的定位（技术岗/操作岗/服务岗/管理岗）
3. AI/自动化对该岗位的冲击程度

### 第 3 步 · 地域校准

根据用户所在区域调整信号：

- 同一个行业在不同区域信号可能不同（如传统重工业在东北是【转】，在动力源是【缩】）
- 参考 strategy/regional/ 的五大区域 × 机会矩阵
- 如果用户在县城，额外参考县域专属机会

### 第 4 步 · 方法论校验

用毛泽东战略思维工具校验判断逻辑：
- 这个方向是真需求还是政策口号？（《关心群众生活》011）
- 该用持久战思维还是速决战思维？（《论持久战》027）
- 主要矛盾是什么？（《矛盾论》018）

## 输出规格

输出一份结构化判断，包含以下模块：

```
## 行业机会扫描结果

### 你的画像
行业：___  岗位：___  地域：___  年龄：___

### 信号判定：【增/转/缩】
（一句话概括 + 确定性星级 ★★★★★）

### 行业大势
（3—5 句话说明这个行业在十五五的处境）

### 你的岗位前景
（这个具体岗位会怎样——升值/贬值/被替代？）

### 地域因素
（在你这个地方，这个行业有什么特殊机会或限制）

### 行动建议
（3 条具体可操作的建议，按优先级排列）

### 风险与红线
（要避开什么坑、什么信号说明该止损）

### 一句话总结
（可记住的一句话）
```

## 测试用例

### 用例 1：产线工人 + 长三角

输入：行业=制造业，岗位=产线操作工，地域=长三角，年龄=35，现状=干了10年

期望信号：【转】
期望要点：
- 行业不消失但操作工会被自动化替代
- 建议路线：考电工证→学PLC→转设备维护（growth_path 线二）
- 长三角制造业密集，技术工人缺口大，升级后就业有保障
- 风险：纯体力操作岗在【缩】，不升级 5 年内被淘汰

### 用例 2：养老护理员 + 县城

输入：行业=养老，岗位=护理员，地域=中部县城，年龄=40

期望信号：【增】★★★★★
期望要点：
- 3 亿老人 + 县城老龄化更重但专业服务几乎空白
- 县域养老是确定性最高的增量赛道
- 建议路线：考护理员证→上门护理→积累口碑→考虑开社区养老服务站
- 风险：体力消耗大、薪酬前期偏低，但长期需求刚性

### 用例 3：K12 教师 + 想转行

输入：行业=教育，岗位=K12学科教师，地域=二线城市，年龄=32，现状=想转行

期望信号：【缩】（K12 学科培训）
期望要点：
- 双减 + 少子化双重打击，K12 学科培训在收缩
- 转型方向：素质教育/成人教育/职业教育/知识付费
- 建议路线：考编进公办 / 转成人技能培训 / 做知识付费个人IP
- 不要恋战在收缩赛道

## 诚实的边界

1. 本 skill 产出的是**方向参考**，不是就业承诺或投资建议。
2. 信号判定基于 2025—2026 年形势，技术变革和地缘冲击可能改变判断。
3. 个体处境受多种因素影响——本 skill 提供的是「看清大势」的工具，不是「替你做决定」的答案。
4. 本 skill 不提供炒股/投资/投机建议，不替代专业职业规划咨询。
