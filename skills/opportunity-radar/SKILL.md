# 机会雷达 skill

## 概述

这个 skill 把 strategy/perspective/ 的六大前瞻主题 + strategy/new_value/ 的十大确定性增量，蒸馏成一个可调用的趋势前瞻服务。用户输入感兴趣的方向，skill 给出 5—10 年的中长期判断。

它解决的问题是：**「未来 5—10 年，大趋势往哪走？哪些机会在形成？哪些「常识」会被打破？」**——不是当下怎么办（那是 industry-scan 的事），是看远一步。

## 什么时候用

- 想判断一个方向是短期热点还是长期趋势
- 考虑进入一个新领域，想知道它 5 年后行不行
- 想了解 AI/机器人/人口/能源对工作的影响
- 想发现「被低估的变量」——大众还没注意但正在改变格局的东西
- 年轻人想长线布局

**不要用于**：短期投机判断、炒股决策、当下行动指南（用 industry-scan）。

## 输入规格

| 参数 | 必填 | 说明 | 示例 |
|---|---|---|---|
| 关注方向 | 是 | 你想看哪个方向的前瞻 | AI影响/机器人/养老/能源/我的行业 |
| 你的行业 | 否 | 便于给出行业特定前瞻 | 嵌入式开发/餐饮/制造业 |
| 时间视野 | 否 | 你想看多远 | 3年/5年/10年 |

## 执行逻辑

### 第 1 步 · 主题映射

把用户的关注方向映射到六大前瞻主题：

| 主题 | 覆盖范围 |
|---|---|
| AI 长波冲击 | AI对百行百业的影响、赢家与输家规律 |
| 人口结构巨变 | 老龄化/少子化/城市化创造和消灭的需求 |
| 能源转型终局 | 化石→新能源的不可逆趋势、后市场机会 |
| 机器人进生产生活 | 工业/服务/人形机器人的成熟度与就业影响 |
| 全球化重构出海 | 世界工厂→世界品牌、模式出海 |
| 技能即服务 | 工作方式本身从岗位变 skill |

### 第 2 步 · 中长期判断

对映射到的主题，给出前瞻判断：

- 趋势总判断（5—10年大方向）
- 中长期赢家与输家（哪些角色崛起/边缘化）
- 被低估的变量（大众还没注意但正在改变格局的）

### 第 3 步 · 确定性评估

如果用户提到了具体行业/岗位，用 new_value 的四维度评估：
- 确定性（政策+市场双驱动？）
- 入场门槛（普通劳动者够不够得着？）
- 窗口期（早期/成长/已拥挤？）
- 抗 AI 性（会被替代还是被放大？）

### 第 4 步 · 持久战校验

用《论持久战》校验：这个趋势是速决战（3年内见分晓）还是持久战（10年+）？给用户现实的时间预期。

## 测试用例

### 用例 1：嵌入式工程师看 AI 影响

输入：关注=AI影响，行业=嵌入式开发

期望前瞻：
- 映射：AI长波冲击 + 技能即服务
- 赢家：能用AI放大自己的专业者、物理世界服务者
- 输家：纯信息处理、标准代码搬运工
- 被低估的变量：经验+AI=超级个体
- 对嵌入式：越靠近物理世界的开发越抗AI替代，你是安全区
- 不确定性：AI技术瓶颈可能让冲击速度比预期慢

### 用例 2：想看养老行业前景

输入：关注=养老，时间视野=10年

期望前瞻：
- 映射：人口结构巨变
- 确定性：★★★★★（4亿老人+家庭照护能力枯竭）
- 赢家：居家护理/银发文旅/健康管理/适老化改造
- 被低估的变量：60后退休潮是第一代有钱有闲有消费力的「新老人」
- 不确定性：退休年龄延迟节奏影响时间线

### 用例 3：想了解机器人行业

输入：关注=机器人

期望前瞻：
- 映射：机器人进生产生活
- 三条赛道不同成熟度：工业（成熟）/服务（成长）/人形（极早期）
- 被低估的变量：机器人的衍生就业——用/修/调/训机器人门槛比想象中低
- 不确定性：人形机器人商业化时间表高度不确定

## 诚实的边界

1. 前瞻给的是**方向**不是**时间表**——知道趋势会来，不预测「具体哪年」。
2. 前瞻基于 2025—2026 年可见变量，技术突破/地缘剧变/黑天鹅可能让判断跑偏。
3. 前瞻不是投资建议/创业承诺，是帮劳动者「看远一步」的参考。
4. 前瞻需要持续校验——变量变了判断要更新。
5. 本 skill 与 industry-scan 的区别：后者看当下（增/转/缩），本 skill 看未来（5—10年趋势）。
