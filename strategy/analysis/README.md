## 一、分析方法论

本解读采用 methodology 中经三年沉淀的**战略文本四步分析法**：
1. **读原文**——逐句引用，不篡改、不删减。

3. **结合时局**——放到 2025—2026 年真实世界检验（中美博弈、科技脱钩、房地产调整、人口负增长、AI 军备竞赛等）。

## 三、全文核心判断（十八篇浓缩）

用 methodology/mao_zedong_thought/reference/ 中的战略思维工具，整部纲要可以浓缩为五个层次：

### 1. 形势判断（第一篇）
「战略机遇和风险挑战并存，不确定难预料因素增多」。风险权重显著上升——大国博弈长期化、外部不确定性增强。
→ 方法论：《矛盾论》（018）抓主要矛盾；《论持久战》（027）相持蓄力。

### 2. 经济引擎（第二—四篇）
实体为本（守制造业比重）、科技自主（六大卡脖子领域决定性突破）、数字赋能（人工智能+全场景）。
→ 方法论：《丢掉幻想，准备斗争》（155）自主攻关；《必须注意经济工作》（008）夯实经济基础。

### 3. 经济循环（第五—七篇）
扩内需（消费+投资）、深改革（统一大市场+财税+国企民企）、稳开放（自主开放+一带一路+人民币国际化）。
→ 方法论：《关心群众生活，注意工作方法》（011）释放内需；《团结起来，划清敌我界限》（181）团结多数。

### 4. 社会底盘（第八—十二篇）
农业强国（粮食绝对安全）、区域协调（三大动力源+战略腹地）、文化强国（文化出海+话语体系）、人口战略（生育友好+银发经济+延迟退休）、共同富裕（分配改革+橄榄型社会）。
→ 方法论：《关心群众生活》（011）解决急难愁盼；《关于正确处理人民内部矛盾的问题》（217）化解社会张力。

### 5. 安全保障（第十三—十八篇）
绿色转型（2030碳达峰）、国家安全（粮食/能源/金融/网络四道防线）、国防（建军百年2027+战略威慑）、民主法治（全过程人民民主+涉外法治）、港澳台（反独反干涉+主导权主动权）、实施保障（党的领导+三个「跟着走」）。
→ 方法论：《一切反动派都是纸老虎》（229）战略藐视；《丢掉幻想，准备斗争》（155）战术重视。

## 五、诚实的边界

1. 本解读是**战略意图分析与时局结合**，不是政策预测，更不是投资建议。规划是「意图」，落地受执行能力、资源约束和外部冲击三重影响。
2. 「逐句」颗粒度按战略密度调整：总论和关键篇做到句段级；程序性表述做节级概括。
3. 时局信息以 2025—2026 年公开形势为准；重大事件后续演变需回头修正。
