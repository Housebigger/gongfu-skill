## 一、本层要解决什么问题

`entrepreneurship/` 层回答了「一个人怎么零成本起步」。但一个人的时间和精力是有上限的——服务化一天最多干 14 小时，内容化一个人做不过来拍摄+剪辑+运营，电商化一个人扛不住选品+客服+发货。

**单干做不出新蛋糕。** README 第二阶段动作链里，「结成协作」是「借势创业」之后的下一个动作——只有把分散的个人力量拧成协作体，才能从「各赚各的小钱」走向「共同做出新的价值」。

本层要回答：**普通劳动者怎么组织起来——合作社、互助小组、技能互换、接包小队、县域集体经济——把分散的劳动力、技能、资源拧成可持续的协作体，做出一个人做不了的事？**

这不是教你「搞人脉」或「混圈子」，而是回答一个具体的问题：**当你的个体生意遇到天花板，或者当一个机会需要多种能力组合而你只有一种时，怎么找到对的人、怎么合作、怎么分钱、怎么不散伙。**

## 二、方法论根基：为什么用毛泽东的战略思维指导协作判断

毛泽东思想中最丰富的遗产之一，恰恰是**如何组织人、如何团结大多数、如何处理集体内部的矛盾**。这些方法论在 worker_guidance 里只用了「个人版」（个人怎么集中优势兵力），本层用的是**「集体版」**——这是方法论资源的盘活。

| 毛泽东著作 | 战略思维工具 | 在协作中的运用 |
|---|---|---|
| 《关于正确处理人民内部矛盾的问题》（217） | 辩证处理内部张力 | 合伙必散的根本原因不是利益冲突（利益冲突是常态），而是**不会处理冲突**。用发展而非对抗解决内部矛盾 |
| 《团结起来，划清敌我界限》（181） | 团结大多数、分清内外 | 找合作伙伴的核心：团结能团结的人，但先划清「能一起干事的人」和「不能一起干事的人」 |
| 《集中优势兵力，各个歼灭敌人》（103） | 聚焦优势、不分散 | 集体版的集中优势兵力——小团队只做一个方向，集中所有人的力量打一个点，不分散 |
| 《关心群众生活，注意工作方法》（011） | 解决群众急难愁盼 | 协作的纽带不是情义，是**一起解决真实的、共同的问题**。利益共同体比感情共同体更牢 |
| 《矛盾论》（018） | 抓主要矛盾 | 协作初期的主要矛盾是「信任未建立」；中期是「分工不均」；后期是「分配不公」。不同阶段处理不同矛盾 |
| 《丢掉幻想，准备斗争》（155） | 丢掉幻想、做最坏打算 | 丢掉「好朋友合伙一定长久」的幻想——**先小人后君子**，把最坏的情况写进协议，反而能长久 |

核心逻辑一句话：**组织人的本质不是画饼，是分好饼——把利益、风险、贡献讲清楚，比任何感情承诺都管用。**

## 三、五种协作形态谱系

协作不是只有「合伙开公司」一种。根据协作的松紧程度，从松到紧排列：

| 形态 | 松紧度 | 核心 | 适合场景 |
|---|---|---|---|
| **技能互换** | 最松 | 我帮你的活，你帮我的活，不涉及钱 | 起步阶段互相补短板 |
| **互助小组** | 松 | 定期交流/共享信息/拼单采购/互推客户 | 同行或同行之间抱团 |
| **接包小队** | 中 | 一个人接不了的大单，几个人组队干 | 跨能力组合接项目 |
| **合伙经营** | 紧 | 共同出资/共担风险/共分利润 | 有稳定业务后正式化 |
| **合作社/集体经济** | 最紧 | 制度化的集体组织，受法律保护 | 农业合作/县域经济/社区服务 |

**一条铁律：从松到紧，逐步升级。** 永远不要一上来就「合伙开公司」——先从技能互换和互助小组开始，跑出了信任再升级到接包小队，业务稳定了再考虑合伙。**信任需要时间验证，不要跳级。**

## 四、信任与分配：怎么避免「合伙必散」

「合伙必散」是中国创业最常见的死法之一。但合伙散伙的真正原因，通常不是「利益冲突」——利益冲突是正常的、必然的。散伙的真正原因是：

1. **事前没谈清楚怎么分钱**——赚了钱才来吵，已经晚了。
2. **贡献没法量化**——你觉得自己干得多，他也觉得自己干得多。
3. **退出机制没设计**——一个人想走，另一个人不让走，僵住了。
4. **用感情替代规则**——「咱们兄弟不用算那么清」——结果恰恰是算不清才散的。

本层的核心方法论：**先小人后君子——把最坏的情况（散伙/亏损/退出）在合作第一天就写下来，反而能避免它发生。**

详见正文文件 `03`。

## 五、跨行业协同：1+1+1 > 3

零成本创业最大的杠杆之一，是**跨能力组合**——一个人只有一种能力，三个人组合就有三种能力，能接一个人接不了的活。

| 组合 | 各自能力 | 组合后能做什么 |
|---|---|---|
| 制造 + 内容 + 电商 | 工厂工人 + 短视频创作者 + 电商运营 | 把工厂产品直接卖向全国 |
| 技术 + 服务 + 运营 | 电工 + 客服 + 推广 | 做一个区域维修服务品牌 |
| 农业 + 内容 + 物流 | 种植户 + 直播带货 + 快递 | 把农产品直销全国 |
| 设计 + 生产 + 渠道 | 设计师 + 工厂 + 销售商 | 做自有品牌产品 |

详见正文文件 `04`。

## 七、诚实的边界

1. 本层是**面向一线劳动者的协作方向参考**，不是法律意见。涉及注册公司/合作社/签协议，务必咨询当地市场监管/法律专业人士。
2. **协作比单干更复杂**——管人比管事难，处理关系比处理业务难。如果你连单干都还没跑通，不要急着合伙。先做好自己的事，自然会遇到对的人。
3. 本层与 `entrepreneurship/` 的关系：后者是「单点突破」，本层是「连点成网」。本层不是 entrepreneurship 的续集，而是它的延伸——当你做大了、遇到天花板了、需要人了，才需要本层。
4. 方法论工具来自毛泽东著作的战略思维方法，不是政治教条。
5. **合规红线极其重要**——涉及钱的协作（集资/分红/吸纳成员），务必了解非法集资、传销、非法用工的法律边界。详见 `05`。
