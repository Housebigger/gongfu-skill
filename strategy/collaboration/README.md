## 三、五种协作形态速查表

| 形态 | 松紧度 | 核心 | 适合阶段 |
|---|---|---|---|
| 技能互换 | 最松 | 互助不涉及钱 | 起步、建立信任 |
| 互助小组 | 松 | 信息/资源共享 | 同行抱团降本 |
| 接包小队 | 中 | 临时组队接大单 | 跨能力组合 |
| 合伙经营 | 紧 | 共同出资共担风险 | 业务稳定后 |
| 合作社/集体经济 | 最紧 | 制度化集体组织 | 农业/社区服务 |

铁律：从松到紧逐步升级，不跳级。每一步至少合作 3—6 个月再考虑升级。

## 五、诚实的边界

1. 本层是**面向一线劳动者的协作方向参考**，不是法律意见。涉及注册/协议/税务，务必咨询当地专业人士。
2. **协作比单干复杂**——如果你连单干都还没跑通，不要急着合伙。先做好自己的事。
3. 方法论工具来自毛泽东著作的战略思维方法，不是政治教条。
4. 合规红线极其重要——涉及钱的协作务必了解非法集资、传销、非法用工的法律边界。
