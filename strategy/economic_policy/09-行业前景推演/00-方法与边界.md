# 09-行业前景推演——方法与边界

日期：2026-06-25
层级：经济政策专栏 Phase 3（宏观议题下沉·逐行业集群·三情景动态推演）

> **核心声明（须先读）**：本层各集群文件均属**情景分析，非预言**。情景给出方向与触发条件，不给时间表、不给点位；不投资建议、不出个股；不政治预测。政策与行业结果之间是观察性关联，非严格因果。本层结论继承 `05-边界与诚实.md` 与 `06-推演方法与外部约束.md` 第四节的全部边界，不替代它们。

## 一、本层定位：宏观议题下沉到产业集群

### 1.1 下沉逻辑

`07`/`08` 两文的六议题推演（内需/消费、房地产、财政与地方债、AI 与新质生产力、外需与出海、人口与劳动力）是**宏观层面**的方向谱——告诉读者大盘在哪几个方向有顺风或逆风。

本层（`09`）承接这份方向谱，将其**下沉到具体产业集群（A–P）**：同样的政策机制，落到绿色能源全链（C）时，触发条件是什么？落到传统重化工与建材（K）时，因果链条和时滞是否不同？对应集群的劳动者，机会窗口和风险节点分别在哪里？

下沉不是重新推演——是把 `07`/`08` 已经建立的情景框架，贴着具体行业的现实进行**精确化和具体化**。

### 1.2 与相邻层的分工

| 层 | 所在位置 | 定位 |
|----|---------|------|
| `perspective/`（主题式中长期前瞻）| `strategy/perspective/` | 主题驱动（如老龄化、AI、绿色）的中长期前瞻，时间跨度 5—15 年，覆盖多个行业截面 |
| `skills/data/industry-signals.yaml`（静态信号卡）| `skills/data/` | 静态截面：增/缩/转三类方向信号，描述产业集群当前位置，不做情景推演 |
| **本层 `09`**（逐集群动态推演）| `strategy/economic_policy/09-行业前景推演/` | **政策机制驱动·逐集群·三情景动态推演**，承接 `07`/`08` 宏观议题，落到具体集群的触发条件、劳动者含义与可证伪点 |

三者互补：`perspective/` 给长线主题方向，`industry-signals.yaml` 给现时位置快照，本层给政策机制驱动下的中期动态情景。阅读某集群前，建议先查 `industry-signals.yaml` 的静态信号，再看本层的动态推演，再参考 `perspective/` 的长线背景。

## 二、下沉方法：单集群推演如何套用六要素

### 2.1 六要素必须全部展开

每个集群文件（如 `C-绿色能源全链.md`、`K-传统重化工与建材.md`）的推演，必须遵循 `06` 第一节的**六要素模板**，六要素不可跳过或合并：

| 要素 | 在集群推演中的具体操作 |
|------|----------------------|
| ① 现状锚点（带来源）| 用可查官方数据锁定该集群当前的规模、就业量、政策环境起点；数据注明截至时间与来源渠道 |
| ② 规律库外推（引 `04`）| **明确引用 `04` 的哪条规律**（时滞分层/传导节点/就业收入类型），再评估该规律在本集群的适用性——有些规律在重资产行业时滞更长，有些在劳动密集型集群的就业传导更直接 |
| ③ 套外部约束 + 国情 | 对照 `06` 第二、三节的共用假设框（中美博弈、外需偏弱、内需修复需时、房地产/地方债压制），判断本集群受哪项约束影响最大，方向为顺风/逆风/待观察 |
| ④ 基准/上行/下行三情景 | 三情景各带：触发条件（可观察经济指标）+ 关键假设 + 对该集群劳动者的方向含义；不给时间表和点位 |
| ⑤ 可证伪点/观察指标 | 列 2—4 个该集群专属的可查指标，区分正向证伪与负向证伪，并明确区分两类负向证伪：(a) 规律前提失效信号（如传导链断裂，须回 ② 重审规律适用性）与 (b) 下行情景触发信号（关键假设朝不利方向兑现，直接切换下行情景路径）；与 `06` 第五节复盘机制衔接 |
| ⑥ 劳动者含义 | 落点在职业/技能/转岗方向，不在资产配置；基准情景下的操作方向 + 下行情景信号出现时的留意点 |

### 2.2 传导节点检查在集群层的执行

引用 `04` 规律后，必须对本集群执行传导节点三检：
- **①财政/信贷是否到位**：政策资金是否已落到本集群（专项债项目是否覆盖此方向、贴息贷款是否在本集群有额度）；
- **②部门分工与量化考核**：本集群有无明确的行业主管部门目标分解（如能耗目标、装机目标、产值目标）；
- **③是否落到具体价格/产能/岗位/收入**：政策传导最终环节——落到工厂开工率、产品出厂价、新增岗位还是劳动力替代，需在集群层逐项判断。

三节点全通则正常外推；有节点断链，在情景中降低该路径确定性，或把"该节点能否打通"列为关键假设。

**负向证伪的两类区分**：若属 (a) 类规律前提失效信号（如传导链断裂：专项债已足额发行但施工未启动），须先回 ② 重审该 `04` 规律是否仍适用于本集群，再决定是否调整情景权重；直接切换下行情景仅适用于 (b) 类——关键假设朝不利方向兑现（详见 `06` §1⑤）。④ 三情景表不能直接处理 (a) 类信号。

### 2.3 显式锚定主驱动的 `07`/`08` 议题

每个集群文件在推演开篇，须用一段话或一张表，显式说明本集群主要受 `07`/`08` 哪个议题驱动（例：绿色能源全链 C 的主驱动是"AI 与新质生产力"+"财政与地方债"两议题（新能源装备属新质生产力政策范畴；大规模算力建设带动电力需求）；传统重化工 K 的主驱动是"内需/消费"+"外需与出海"+"财政与地方债"）。

这一步是验证"下沉"不是"另起炉灶"的锚点：本集群的情景方向，须能被追溯回 `07`/`08` 的宏观情景框架。若推演方向与宏观议题方向矛盾，须显式说明原因（如集群受特定产业政策保护而与宏观趋势部分脱钩），而不是悄悄绕过。

### 2.4 双时间视角的集群化

继承 `06` 第一节②的双时间视角，但在集群层做行业化处理：
- **近期视角（2026—2027）**：本集群在此窗口的政策支撑强度如何？用工高峰或收缩压力大概率在哪个时段？
- **十五五期视角（2026—2030）**：本集群在"双碳/新质生产力/共同富裕"主线中的结构性位置是什么？长周期是顺风还是逆风？

两个视角须分开表述，不混用。

## 三、逐行业推演专属诚实边界

本层各集群文件均受以下六条边界约束，且**继承 `06` 第四节与 `05` 的全部边界，不替代它们**。

### 3.1 边界一：情景非预言

本层集群文件给出的三情景，是"若 A（触发条件）+ B（关键假设）成立，则该集群方向为 C"的条件式表述，不是"某年某集群将发生某事"的时间表表述。读者见到某情景条件在现实中开始兑现，是参考信号，不是确定性结论。

### 3.2 边界二：给方向不给时间表

集群文件不写"2027 年某行业产能将达多少"或"某职业岗位将增加 X 万个"。方向是"顺风/逆风/不确定"，时间节点只能是"若触发条件于某时段出现"，不给具体年份点位。

### 3.3 边界三：不构成投资建议，不出个股

集群推演的落点是"劳动者的职业/技能/转岗方向"，不是"哪个板块/哪支股票值得买"。集群中不出现证券代码、股价预判、仓位建议。如需金融资产配置参考，请咨询持牌金融机构。

### 3.4 边界四：不做政治预测

集群文件不预测政策走向背后的政治动机，不做党政人事分析，不超出已公开官方文件（政府工作报告、五年规划、行业主管部门文件）的范围推断政治意图。

### 3.5 边界五：政策↔行业结果是观察性关联，非严格因果

每个集群都在政策之外同时受市场需求、技术变化、国际竞争、产业周期多重因素影响。本层描述的"政策机制→集群影响"是观察性关联，不是受控实验下的因果推断。行文遵守"伴随""在……政策落地后观察到""与……政策同期出现"等措辞，不使用"由于政策……必然导致"的强因果表述。

### 3.6 边界六：本层结论继承 `06`/`05` 的全部边界，不替代它们

`05-边界与诚实.md` 是专栏总边界，`06` 第四节是 Phase 2 推演边界，本节是 Phase 3 集群层专属边界。三者共同构成本专栏的完整诚实框架，层层叠加，不互相覆盖。阅读任一集群文件前，建议先阅读 `05`、`06` 第一节（六要素操作手册）和 `06` 第四节，再读本节；集群复盘照 `06` 第五节机制执行。

### 4.3 情景假设的定期更新

按 `06` 第五节复盘机制，本层各集群文件的情景假设与触发条件须定期更新：

- **每半年轻量复盘**（建议 7—8 月 / 1—2 月）：检查各集群主驱动的 `07`/`08` 宏观情景是否发生切换，相应更新集群层的触发条件权重；
- **每年深度复盘**（政府工作报告发布后 3—4 月）：结合新的政府工作报告，更新各集群的产业政策锚点，必要时修订可证伪点；
- **非周期性紧急更新触发条件**：与 `06` 第五节一致——中美关税协议发生根本性变化、重大地缘冲击影响能源价格或供应链、特定集群出现系统性风险信号（如大规模产能退出、行业级政策急转）、国家年度 GDP 增速出现重大偏离（实际与目标差距扩大，可能触发 `07`/`08` 情景切换）。

落地格式：在集群文件头部添加或更新一行标注，格式为`当前情景倾向（YYYY-MM 复盘）：倾向[基准/上行/下行]情景，原因简述。`，以便翻阅时直接可见最新研判。
