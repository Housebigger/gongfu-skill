# 09-行业前景推演——逐集群政策机制驱动情景推演

日期：2026-06-25

---

## 本层定位

本子层把 `07`/`08` 的宏观议题推演**下沉**到具体产业集群：以 06 六要素法为框架，结合行业政策锚点（原料库 `policy_archive/03`）与各集群的政策传导规律，对每个集群给出基准/上行/下行三情景，并标注触发条件、关键假设、可证伪点与劳动者含义。

> 情景为方向研判，非预言。给方向不给时间表，不含个股推荐，不作政治预测。数据带官方来源，先搜后确认。

---

## 与相邻层的分工

| 层 | 性质 | 与本层的关系 |
|----|------|-------------|
| `strategy/economic_policy/07`、`08` | 宏观议题推演（内需/新动能/出海等六大议题） | 本层把这些宏观判断"翻译"到具体产业集群 |
| `strategy/perspective/` | 行业主题中长期前瞻（5—10 年视角，主题式） | 前瞻给方向但不按集群推演情景；本层按集群给三情景，视角更近、更具体 |
| `skills/data/industry-signals.yaml` | 静态信号卡（集群级别的政策/技术/就业信号，evergreen） | 静态信号卡是稳定结构化工具；本层是基于当前政策形势的情景推演，需按复盘机制定期更新 |
| `strategy/references/policy_archive/` | 原料层（政策原文/数据溯源） | 本层引用 `00-信源清单.md` 与 `03-行业政策锚点-六集群.md` 作为直接原料 |
