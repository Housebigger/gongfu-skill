日期：2026-06-15

## 三、四条路径速查表

| 路径 | 核心逻辑 | 适合谁 | 见效速度 | 天花板 |
|---|---|---|---|---|
| 服务化 | 把已有手艺直接卖给客户 | 有手艺的技术工人/服务业从业者 | 最快（第一周可能接单） | 受限于个人时间 |
| 内容化 | 用经验做内容获客，再变现 | 有 3 年+经验和表达意愿的人 | 最慢（1—3 个月） | 最高（规模效应） |
| 电商化 | 卖货但不囤货 | 有选品眼光和运营能力的人 | 中等（2—4 周） | 高（可规模化） |
| 技能出租 | 把闲置设备/资质/场地/时间租出去 | 有可出租资源的人 | 快（1—2 周） | 受限于资源量 |

阅读建议：先读 `00`（框架），再按你最可能走的路径读对应正文（01—04），然后读 `05`（AI 工具）和 `06`（节奏与红线），最后务必读 `07`（诚实劝退）。

## 四、三种阶段标记

贯穿本层，帮你判断自己处于哪个阶段：

- **【试】验证阶段**——还没赚钱，用最小成本试路。目标：卖出第一单。不辞职、不借钱。
- **【稳】起步阶段**——有稳定小额收入，模式跑通。目标：提升稳定性、扩大客源。仍不辞职，直到副业稳定超主业 3 个月。
- **【扩】发展阶段**——收入可替代主业。目标：团队化/品牌化。少数人的选择，不强求。

## 六、诚实的边界

1. 本层是**面向一线劳动者的零成本创业方向参考**，不是创业承诺，更不是投资/融资建议。
2. 「零成本」指不负债/不囤货/不租铺，不是「保证赚钱」。创业永远有风险，本层做的是把风险压到最低。
3. 案例基于 2025—2026 年真实商业模式，隐去个人身份。平台规则和政策会变，读者应持续校准。
4. 方法论工具来自毛泽东著作的战略思维方法，不是政治教条。
5. 与 `worker_guidance/` 交叉引用不重复：后者是「某行业怎么办」的纵深，本层是「怎么从打工变成自己的事」的横向方法论。
