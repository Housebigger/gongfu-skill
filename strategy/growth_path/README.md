# 个人成长地图层——总索引

日期：2026-06-15

本层是 strategy 的第六个子模块，对应 README 第二阶段动作链中的「顺势成长」。它回答的问题是：**站在你现在的人生位置上，作为一线劳动者，该怎么一步步成长。**

---

## 一、与现有层的关系

| 层 | 切片维度 | 回答的问题 |
|---|---|---|
| `worker_guidance/` | 行业切片 | 某行业的一线劳动者怎么办 |
| **`growth_path/`（本层）** | **个人切片** | **某个人生阶段的人怎么走** |
| `entrepreneurship/` | 行动模式切片 | 怎么从打工变成自己的事 |

本层与 worker_guidance 互补：后者按「行业」切，本层按「人生阶段」切。两者交叉使用效果最好。

## 四、学习地图三层结构（贯穿全层）

```
第三层 · 判断力/经营能力  ← entrepreneurship + collaboration 层
        （从「会干活」到「会经营」）
              ▲
              │ 长期积累
        ──────┴──────
第二层 · 证书/资质  ← 能力的社会证明
        （有手艺+有证书 = 完整竞争力）
              ▲
              │ 6个月—2年
        ──────┴──────
第一层 · 手艺  ← 能直接换钱的技能
        （有手艺就有饭碗——最底层安全网）
              ▲
              │ 1—3年
        ─────────────
```

铁律：从下往上盖楼。先有手艺，再考证书，再学经营。不能跳级。

## 六、诚实的边界

1. 本层是**面向一线劳动者的个人成长参考**，不是人生/职业规划的权威答案。
2. 四种画像是典型分类，你可能处于两种之间——各取所需。
3. 年龄段是参考区间——30 岁不一定比 22 岁晚，45 岁不一定比 30 岁迟。
4. 资源信息基于 2025—2026 年，平台/政策会变，请核实最新情况。
5. 方法论工具来自毛泽东著作的战略思维方法，不是政治教条。
