# 产业链分析提炼层——总索引

日期：2026-06-23

本层是战略库的**第二核心根源**的人读提炼层。

第一根源（十五五规划）给劳动者一张**国家产业方向的宏观地图**——哪些行业被国家力推、哪些领域有政策红利。但宏观地图有盲点：它告诉你方向，却不告诉你一条具体产业链的"钱"在哪里、"难度"在哪里、你的岗位是卡点还是易替代的边缘。

本层填补这一盲点。它把产业链研究者 Serenity（白毛股神）的**卡点分析法**剥离个股操作，翻译成劳动者的**行业判断 / 趋势前瞻 / 创业切入工具**。一宏一微，互相补位：

| 根源 | 视角 | 回答的问题 |
|------|------|------------|
| 第一根源：十五五规划 | 宏观·国家 | 国家在推哪些产业？我在对的赛道吗？ |
| 第二根源：Serenity产业链分析法 | 微观·产业链 | 这条链里哪个环节最难替代？我的位置卡不卡？ |
