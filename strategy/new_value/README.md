日期：2026-06-15

## 三、新蛋糕三种类型速查

| 类型 | 「新」在哪 | 典型机会 |
|---|---|---|
| 新需求 | 以前不存在的需求 | 银发陪伴/宠物殡葬/低空旅游/情绪价值服务 |
| 新供给 | 新的满足方式 | AI 辅助内容/即时零售/绿色能源后市场 |
| 新组织 | 新的连接方式 | 跨境直连/农产品直销/社区团购 |

## 五、诚实的边界

1. 本层是**机会地图**，不是投资建议/创业保证。
2. 窗口期会关闭，饱和会发生——读者应持续校准。
3. 本层是索引层——怎么做见 `entrepreneurship/`，哪个行业见 `worker_guidance/`。
4. 方法论工具来自毛泽东著作的战略思维方法，不是政治教条。
