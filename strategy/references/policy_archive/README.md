# policy_archive — 国家经济政策原文源料层

## 定位

本目录是 `strategy/economic_policy/` 专栏的**原文源料层**（第三条研究根源），与同级的两条根源并列：

| 根源 | 路径 | 性质 |
|------|------|------|
| 第一根源 | `references/十五五规划纲要-全文.md` | 顶层规划全文 |
| 第二根源 | `references/serenity/` | 行业/趋势沉淀 |
| **第三根源（本目录）** | `references/policy_archive/` | 国家经济政策原文与数据溯源 |

本目录**只做记录，不做评价**：收录政策原文摘要、官方统计数据线索、核心会议公报，供 `strategy/economic_policy/` 复盘提炼时作为第一手引用依据。

## 诚实声明

- **公开官方源为主**：所有条目以可公开访问的政府网站、官方媒体为来源，每篇标注来源 URL 和抓取/整理日期。
- **抓不到的只登记 URL，不伪造内容**：凡网络抓取失败（404、访问限制）或内容无法在线获取的，在信源清单中标注 ⚠仅登记，原文内容处注明"未能抓取，以下为已核实公开事实整理"，并逐条注明来源。
- **无投资建议，无股票代码**：本目录不含任何证券代码、股票推荐或投资建议，内容面向一线劳动者的政策认知。
- **不作政治评价**：仅记录政策内容与可核实的官方结果数据，不作政治立场表态或倾向性解读。

## 信源总览

详见 [`00-信源清单.md`](./00-信源清单.md)。

信源分两类：
- **政策原文库**：国务院、中国政府网、国家发改委、共产党员网等官方平台
- **真实结果数据**：国家统计局发布的统计数据，以及官方回顾性分析文章

`00-信源清单.md` 已在原有宏观政策信源基础上**扩充"行业政策信源"段**，收录六大集群（绿色能源/重化工与建材/民生服务/基建物流房地产/先进制造与硬科技/农业与乡村振兴）的主要行业政策来源，供 `09-行业前景推演/` 各集群文件溯源使用。
