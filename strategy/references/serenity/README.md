# Serenity 原文源料层

## 这是什么

本目录是 **Serenity（网名"白毛股神"）产业链分析法** 的原文源料层，是战略库第二核心根源的"源"，地位对应第一根源的 `strategy/references/十五五规划纲要-全文.md`。

Serenity 是活跃于 X（前 Twitter）的产业链研究者，其方法论以"瓶颈理论"/"紫苏叶理论"著称：不追热门龙头，而是在产业链中定位最难替代、最易断供的物理卡点环节，在机构资金前布局。本目录收录其公开可及的方法论文献，供后续提炼使用。
