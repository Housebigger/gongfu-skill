# semiconductor_outlook/ — 华为韬（τ）定律 → 半导体产业发展逻辑与方向

## 专题定位

以华为韬（τ）定律为锚定，研究半导体产业的发展逻辑与未来方向，最终落到一线劳动者——半导体制造/封测/设备/材料/设计链条上的工人、工程师、求职者——的岗位卡位与技能选择。

本专题是技术/产业逻辑驱动（政策为背景），不替代政策主线分析。

## 诚实边界

本专题采用**三分陈述法**：① 事实（有官方来源）/ ② 厂商主张与行业共识（需独立验证）/ ③ 远期预测（情景化，非预言）。预测给方向、不给时间表，并附可证伪点。

- 原始信源与权威解读见原料库 `../references/huawei_tau/`
