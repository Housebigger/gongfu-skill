# supportive_materials

This section stores Hermes project support artifacts that are not part of the main methodology note taxonomy.

## Subfolders

- `system_settings/`
  - environment and machine-level support files
- `experience_crystallization/`
  - distilled execution wisdom, workflow lessons, and reusable practical experience gathered while doing real tasks
  - start from `experience_crystallization/100-theme-summaries-index.md` for the new cross-article theme summaries
  - keep `experience_crystallization/README.md` as the dual-layer navigator for theme summaries and legacy stage summaries
- `spiritual_growth/`
  - persona self-description, reflective growth notes, and project-facing inner development logs written in the course of long-term study

## Writing principle

Files here support the main exploration work:
- methodology outputs belong in `methodology/`
- durable execution insights and reflective companion materials belong in `supportive_materials/`
